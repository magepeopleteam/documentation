EVT‑100

Event Booking

Sell tickets, manage capacity, and run check‑in for conferences, concerts, and multi‑session events.

Addons05
  • Draw custom venue maps, assign seat tiers, and block off rows for accessibility or holds. Pulls live availability into the booking flow. Read the Seating Chart Designer docs →
  • Bundle several sessions or days into a single pass with its own price and entry rules, separate from single‑day tickets. Read the Multi-Day Pass Builder docs →
  • Automatically queues people once an event sells out and releases seats to the next person in line when a ticket is cancelled. Read the Waitlist Manager docs →
  • Turns any phone into a gate scanner. Marks tickets as used in real time and flags duplicates instantly. Read the QR Check‑In Scanner docs →
  • Adds a tiered sponsor wall to the event page and confirmation emails, with click tracking per sponsor. Read the Sponsor Showcase docs →
RNT‑200

Equipment Rental

Run a rental counter for gear, tools, or venue equipment with deposits, availability, and return tracking built in.

Addons05
  • Holds a refundable deposit at checkout and releases or partially withholds it after the item is inspected on return. Read the Damage Deposit Handler docs →
  • Two‑way sync with Google and Outlook calendars so rental blocks stay accurate across every channel you book from. Read the Availability Calendar Sync docs →
  • Draw delivery radii on a map and charge distance‑based fees automatically at checkout. Read the Delivery Zone Mapper docs →
  • Offers an optional protection plan at checkout that waives part of the deposit liability for covered renters. Read the Rental Insurance docs →
  • Charges a configurable hourly or daily fee automatically once an item passes its scheduled return time. Read the Late Return Fees docs →
BUS‑300

Bus Booking

Sell seats across fixed routes, manage fleets and drivers, and give riders live updates on where their bus is.

Addons05
  • Build seat maps per coach type, including aisle gaps, double seats, and accessibility rows. Read the Seat Layout Designer docs →
  • Define routes with ordered stops, per‑leg pricing, and transfer points between connecting services. Read the Route & Stop Manager docs →
  • A lightweight console for assigning drivers to trips, swapping coverage, and pushing schedule changes to driver phones. Read the Driver Dispatch Panel docs →
  • Streams driver app location to a rider‑facing map so passengers can see exactly when their bus will arrive. Read the Live GPS Tracking docs →
  • Applies automatic price breaks when a single booking reaches a configurable number of seats. Read the Group Discount Rules docs →
TRV‑400

Travel & Tours

Package multi‑day trips with guides, pricing in local currency, and bundled extras like transport or meals.

Addons05
  • Lay out a trip day by day with activities, free time, and meeting points, shown to travelers as a readable timeline. Read the Itinerary Builder docs →
  • Displays and charges prices in the traveler's local currency using daily exchange‑rate updates. Read the Multi‑Currency Pricing docs →
  • Assigns guides to departures based on language, availability, and group size, with automatic conflict warnings. Read the Guide Assignment docs →
  • Combines flights, lodging, and activities into a single bookable package with one combined price. Read the Package Bundler docs →
  • Offers optional trip‑cancellation and medical coverage at checkout through a connected insurance provider. Read the Travel Insurance docs →
CAR‑500

Car Rental

Manage a fleet of vehicles for self‑drive hire, with verification, mileage, and fuel rules handled automatically.

Addons05
  • Tracks every vehicle's status, service history, and current location across branches in one dashboard. Read the Fleet Manager docs →
  • Requires a driver's license upload at booking and checks expiry and minimum age before confirming the rental. Read the License Verification docs →
  • Logs odometer readings at pickup and return, and bills overage automatically against the plan's included mileage. Read the Mileage Tracker docs →
  • Lets a renter register a second driver on the agreement for a flat fee, with the same license checks applied. Read the Additional Driver docs →
  • Compares the fuel level at pickup and return and applies the correct charge for full‑to‑full or prepaid policies. Read the Fuel Policy Calculator docs →
