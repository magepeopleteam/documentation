wpevently

Event Manager PluginWpEvently

Sell tickets, manage capacity, and run check‑in for conferences, concerts, and multi‑session events.

Addons14
  • The event max-min addon is necessary for limiting the amount of tickets sold. Organizers can set a maximum buying quantity or minimum purchase quantity.
  • This is event marketplace addon that is needed to allow event submit from frontend. Multiple organisers can sell tickets together in a same site.
  • QR code addon is needed to validate the ticket. If you have a large event then this addon must need to check ticket validity by scanning by QR scanner or mobile.
  • Waitlist addon is very useful, if any ticket became sold out organiser can get subscription for next available ticket sell.
  • Event Calendar addon is a nice enhancement of event list display in calendar so that everyone can easily understand event date.
  • This Addon is mainly used for admin, if anyone wants to get order offline like phone order then after taking order admin can add attendee from dashboard.
  • Send automatic reminder emails before an event. This helps reduce no-shows and keeps attendees informed.
  • Enable customers to purchase multiple tickets in groups with flexible quantity selection options.
  • A seat plan addon is needed for that organiser who wants to display a seat plan for customers to choose seating during event ticket buying.
  • A membership pricing addon for the event is needed for that organiser who wants to offer different pricing based on the user role of a WordPress website.
  • Duplicator addon mainly needed to save time while event creating, if you have an already created event and you want to duplicate that event then this addon needed.
  • Assign different ticket types based on categories for better organization and flexible event pricing options.
  • Event coupon addon that is used for marketing purpose and giving discount to customer for boosting sell.
  • The Event Review and Rating Addon is a fantastic value-added offering designed to help businesses gather feedback from customers about their events.
wpbusticketly

Bus Booking ProWpBusTicketly

Sell seats across fixed routes, manage fleets and drivers, and give riders live updates on where their bus is.

Addons05
  • Build seat maps per coach type, including aisle gaps, double seats, and accessibility rows.
  • Define routes with ordered stops, per‑leg pricing, and transfer points between connecting services.
  • A lightweight console for assigning drivers to trips, swapping coverage, and pushing schedule changes to driver phones.
  • Streams driver app location to a rider‑facing map so passengers can see exactly when their bus will arrive.
  • Applies automatic price breaks when a single booking reaches a configurable number of seats.
wptravelly

Tour Booking ProWpTravelly

Package multi‑day trips with guides, pricing in local currency, and bundled extras like transport or meals.

Addons05
  • Lay out a trip day by day with activities, free time, and meeting points, shown to travelers as a readable timeline.
  • Displays and charges prices in the traveler's local currency using daily exchange‑rate updates.
  • Assigns guides to departures based on language, availability, and group size, with automatic conflict warnings.

wpbookingly

Service Booking ProWpBookingly

Schedule appointments for salons, clinics, repair shops, or any service business with staff and resource availability built in.

Addons05
  • Assigns appointments to specific staff or rooms based on working hours, skills, and existing bookings.
  • Lets customers book a weekly or monthly series in one step instead of repeating the same booking each time.
  • Sends automatic reminders before an appointment, with timing and channel configurable per service.

Taxi Cab and Chauffeur Booking ProeCab

Run on‑demand and pre‑booked rides with fare estimation, driver dispatch, and corporate billing.

Addons05
  • Quotes a fare before booking using distance, time of day, and configurable surge zones.
  • Pushes ride assignments straight to a driver's phone and syncs status back in real time.
  • Matches incoming ride requests to the nearest available driver and reassigns automatically if one declines.

wprently

Booking and Rental Manager ProWpRently

Run a rental counter for gear, tools, or venue equipment with deposits, availability, and return tracking built in.

Addons05
  • Holds a refundable deposit at checkout and releases or partially withholds it after the item is inspected on return.
  • Two‑way sync with Google and Outlook calendars so rental blocks stay accurate across every channel you book from.
  • Draw delivery radii on a map and charge distance‑based fees automatically at checkout.

Car Rental Plugin for WordPress

Manage a fleet of vehicles for self‑drive hire, with verification, mileage, and fuel rules handled automatically.

Addons05
  • Tracks every vehicle's status, service history, and current location across branches in one dashboard.
  • Requires a driver's license upload at booking and checks expiry and minimum age before confirming the rental.
  • Logs odometer readings at pickup and return, and bills overage automatically against the plan's included mileage.
