Installation & Setup

Install the Car Rental Manager plugin via WordPress admin, zip upload, or FTP, then use the setup wizard to configure your car rental system.

Plugin Installation

  • Install via WordPress admin: Plugins > Add New, search for "Car Rental Manager"
  • Upload the Pro version zip from MagePeople after purchase
  • WooCommerce integration for secure payment processing
  • Works with or without WooCommerce (own PayPal/Stripe)

Setup Wizard

  • Run the guided setup wizard upon activation
  • Configure labels, slug, and basic preferences
  • Set currency, timezone, and default rental duration
  • Get pre-built car rental items to start quickly

Shortcode Guidelines

Display car rental booking forms and vehicle listings on any page using dedicated shortcodes with configurable parameters.

Booking Form Shortcodes

  • [mpcrbm_booking form='inline' progressbar='no']
  • [mpcrbm_booking form='horizontal' progressbar='yes']
  • Choose from inline, horizontal, or vertical form styles
  • Enable or disable the booking progress bar

Vehicle Listing Shortcodes

  • [rent-list type='car'] for car-specific listings
  • [rent-list style='grid'] for grid display
  • [rent-list style='list'] for list display
  • Filter by category, order, and number of items
