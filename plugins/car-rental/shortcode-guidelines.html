Shortcode Guidelines

Car Rental Manager provides shortcodes to embed booking forms, vehicle listings, and search interfaces on any WordPress page or post. Each shortcode supports configurable parameters for layout, filtering, and behaviour.

Booking Form Shortcodes

Use the [mpcrbm_booking] shortcode to display the rental booking form. Multiple layout styles and options are available.

Form Layouts

  • [mpcrbm_booking form='inline' progressbar='no'] — Inline form without progress bar.
  • [mpcrbm_booking form='horizontal' progressbar='yes'] — Horizontal step layout with progress indicator.
  • [mpcrbm_booking form='vertical' progressbar='yes'] — Vertical stacked form with progress steps.
  • [mpcrbm_booking form='inline' progressbar='yes' show_category='yes'] — Inline form with category selection.

Form Parameters

  • form — Layout style: inline, horizontal, or vertical.
  • progressbar — Show step progress: yes or no.
  • show_category — Include category filter: yes or no.
  • show_vehicle — Display vehicle selection step: yes or no.
  • class — Additional CSS class names for custom styling.

Vehicle Listing Shortcodes

Display your fleet with the [rent-list] shortcode. Supports grid and list views with filtering options.

List Display Options

  • [rent-list type='car' style='grid'] — Car fleet in grid layout.
  • [rent-list type='car' style='list'] — Car fleet in list layout.
  • [rent-list type='car' style='grid' order='asc'] — Ascending order by title.
  • [rent-list type='car' style='grid' order='desc'] — Descending order by title.

List Parameters

  • type — Post type to list: car.
  • style — Display style: grid or list.
  • order — Sort order: asc (ascending) or desc (descending).
  • category — Filter by category slug (e.g., suv, sedan).
  • posts_per_page — Number of vehicles to display (default: all).
  • show_pagination — Enable pagination: yes or no.

Search & Filter Shortcodes

Provide advanced search capabilities with dedicated filter shortcodes.

Search Form Shortcode

  • [mpcrbm_search placeholder='Search vehicles…'] — Basic search input.
  • [mpcrbm_search style='compact'] — Compact search bar.
  • [mpcrbm_search style='full' show_category='yes'] — Full search with category dropdown.
  • Results update dynamically via AJAX on supported themes.

Category Filter Shortcode

  • [mpcrbm_categories style='dropdown'] — Dropdown category selector.
  • [mpcrbm_categories style='list'] — Checkbox-style category list.
  • [mpcrbm_categories style='buttons'] — Button group filter.
  • Combine with [rent-list] for interactive filtering.

Special Purpose Shortcodes

Dashboard & Account

  • [mpcrbm_dashboard] — Customer dashboard showing past and upcoming bookings.
  • [mpcrbm_my_account] — Account management area for profile and booking history.
  • Both shortcodes require the user to be logged in.

Location & Extras

  • [mpcrbm_locations] — Display available pickup/drop-off locations.
  • [mpcrbm_extras] — Show extra services available for add-on purchase.
  • [mpcrbm_faq] — Render the FAQ section for customer self-service.

Shortcode Best Practices

  • Place only one booking form shortcode per page to avoid conflicting instances.
  • Use a dedicated page for the fleet listing with [rent-list] for better SEO.
  • Always test shortcodes in the WordPress customizer or a staging site before going live.
  • Combine shortcodes with Gutenberg blocks or page builders for advanced layouts.
  • Wrap shortcodes in custom containers if you need additional CSS styling.
