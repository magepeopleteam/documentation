Shortcode Guidelines

Use shortcodes to display rental items, search forms, and booking widgets anywhere on your WordPress site. All shortcodes support a consistent set of parameters for styling, filtering, and pagination.

Rent List Shortcode

The primary shortcode for displaying rental items is [rent-list]. It renders a grid or list of rental products filtered by type, category, or custom query.

Shortcode Parameters

The following table lists all supported parameters for the [rent-list] shortcode.

Parameter Type Default Description
style string grid Display layout: grid or list
show integer 6 Number of items to display per page
order string DESC Sort order: ASC (ascending) or DESC (descending)
orderby string date Sort field: date, title, price, rand, menu_order
type string — Filter by rental type: bike, car, resort, equipment, dress; or bike_car_sd for single-day bike/car, bike_car_md for multi-day
category string — Comma-separated list of category IDs or slugs to filter by
ids string — Comma-separated list of specific product IDs to display
columns integer 3 Number of grid columns (1–6). Only applies when style='grid'
pagination string on Show pagination: on or off
search string off Show a search bar above the list: on or off
class string — Additional CSS class names for the wrapper div

Usage Examples

Basic Grid

  • [rent-list] — default grid with 6 items
  • [rent-list style='grid' columns='4' show='12'] — 4-column grid with 12 items
  • [rent-list style='list' show='10'] — list layout with 10 items

Filtered by Type

  • [rent-list type='bike'] — only bike rentals
  • [rent-list type='car' order='ASC' orderby='title'] — cars sorted alphabetically
  • [rent-list type='bike_car_sd'] — single-day bikes and cars
  • [rent-list type='equipment' show='8' pagination='off'] — equipment without pagination

Filtered by Category

  • [rent-list category='15'] — items in category ID 15
  • [rent-list category='mountain-bikes,road-bikes'] — items in slug-matched categories
  • [rent-list ids='42,58,73'] — display only those three products
  • [rent-list category='12' type='bike'] — combined filters

With Search

  • [rent-list search='on'] — adds a live-search input above the list
  • [rent-list search='on' style='grid' columns='3'] — searchable grid
  • [rent-list search='on' type='car' show='20'] — searchable car list with 20 items

Additional Shortcodes

Rent Add to Cart

  • [rent-add-to-cart id='1'] — single-item add-to-cart button
  • Useful for direct purchasing of a specific rental product
  • The id parameter accepts a valid WooCommerce product ID

Rent Search Form

  • [rent-search-form] — renders the advanced rental search form
  • Includes date picker, type selector, and category dropdown
  • Results can be displayed on a separate search results page
  • Customizable form fields via the plugin settings

Rent Cart

  • [rent-cart] — displays a mini rental cart for the quick checkout flow
  • Shows selected items, dates, and total cost
  • Works with the plugin's standalone checkout mode

Rent Checkout

  • [rent-checkout] — renders the standalone rental checkout form
  • Collects customer details, rental dates, and payment info
  • Only applicable when using the plugin's built-in checkout
