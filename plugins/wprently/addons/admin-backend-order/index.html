Admin Backend Order Add‑On

Create and manage rental orders directly from the admin dashboard. Place bookings manually for customers and maintain full control over rental operations.

Installation

  1. Download the addon zip from your MagePeople account.
  2. Go to Plugins → Add New → Upload Plugin.
  3. Upload the zip file and click Install Now.
  4. Activate the plugin from the Plugins page.

Usage

  1. After activation, a new Backend Order section appears in the WpRently admin menu.
  2. Select a rental item, customer, and rental dates.
  3. Configure any addon options or extras.
  4. Submit the order — it is processed through WooCommerce and a confirmation email is sent to the customer.
