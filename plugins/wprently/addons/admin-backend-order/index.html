Admin Backend Order Add‑On

What if your customers are away from a PC or a smartphone and can only request you to make an order for them? The backend order addon allows admins to create and manage rental orders directly from the WordPress dashboard on behalf of customers.

Installation

  1. Visit the Backend Order Addon page and purchase the addon.
  2. You will receive a download link after purchase.
  3. Go to your WordPress dashboard and navigate to Plugins → Add New → Upload Plugin.
  4. Upload the downloaded zip file and click Install Now.
  5. After installation, click Activate.

Usage

  1. After activation, look for the Purchase Order option in the WpRently admin menu.
  2. Open Purchase Order — all rental items are displayed with Edit, View, and Booking Order options.
  3. Select a rental item and click Booking Order.
  4. Instead of a front-end booking system, you get an order panel on your backend.
  5. Configure the rental item, customer details, and rental dates, then submit the order.
  6. The order is processed through WooCommerce and a confirmation email is sent to the customer.
