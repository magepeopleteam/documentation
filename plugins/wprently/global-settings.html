Global Settings

Configure your rental system globally from the WordPress admin. Navigate to WpRently → Settings to access the settings panel with the following tabs.

1. General Settings

Configure all general configuration for your rental items, including labels, slugs, checkout preferences, and display options.

Labels & Slugs

  • Rental Label: Change the front-end display name for rentals
  • URL Slug: Set the base slug for rental pages
  • Category Base: Customize the category URL prefix
  • After changing slugs, flush permalinks by saving Settings → Permalinks

Checkout Mode

  • WooCommerce Checkout: Uses standard WooCommerce cart and checkout
  • Quick Checkout: Built-in lightweight checkout for faster rentals
  • Guest Checkout: Allow rentals without requiring a user account
  • Terms & Conditions: Require agreement to rental T&Cs at checkout

Display Settings

  • Display Item Available Quantity: Toggle to show stock quantity box on the front end
  • Enable Shipping: Toggle shipping for rental services (configure zones in WooCommerce)
  • Add to Cart Shortcode: Copy and paste the shortcode to display add-to-cart anywhere

2. Translations Settings

Change field labels into your own language directly from the settings panel.

  • All front-end labels can be customized (search form, booking form, buttons, etc.)
  • Changes are stored as site options, not translation files

3. Style Settings

Change the color of the rental system globally.

  • Customize primary, secondary, and accent colors
  • Changes apply to all shortcodes and front-end templates

4. Custom CSS

Add custom CSS code globally.

  • Apply custom styles to all rental pages and shortcodes
  • Overrides theme and plugin default styles

5. Payment Settings

Configure the payment system for your rental service.

  • Choose between WooCommerce gateways or standalone PayPal/Stripe
  • Configure deposit requirements and payment methods

6. PDF Settings

Configure the PDF invoice for rental bookings.

  • Customize invoice layout, logo, and address details
  • Auto-generate PDF invoices on order completion

7. Email Settings

Configure email notifications for your rental service.

  • From Name: Sender name for all rental-related emails
  • From Address: Sender email address for rental notifications
  • Email Type: Choose between HTML, Plain Text, or Multipart
  • CC & BCC: Add additional recipients for booking confirmations

8. Report Settings

Control which fields appear on reports for the rental service.

  • Enable or disable specific data fields on rental reports
  • Customize the report layout to show relevant metrics

9. Super Slider Settings

Configure slider settings for rental item image galleries.

  • Set slider transition effects, speed, and autoplay options
  • Configure thumbnail navigation and lightbox settings

10. License

Add and manage your license key for the Pro version.

  • Enter your license key to receive automatic updates
  • Check license status and renewal information
