Global Settings

Configure the plugin globally from the WordPress admin. All settings are located under WpRently → Settings and apply across your entire rental catalogue.

General Settings Tab

Core plugin configuration including labels, slugs, and checkout preferences.

Labels & Slugs

  • Rental Label: Change the front-end display name for rentals (e.g., "Rentals", "Bookings", "Hire Items")
  • URL Slug: Set the base slug for rental archive pages (default: /rentals/)
  • Category Base: Customize the category URL prefix (default: /rental-category/)
  • Type Labels: Rename each rental type label individually (Bike, Car, Resort, etc.)
  • After changing slugs, flush permalinks by saving Settings → Permalinks

Checkout Mode

  • WooCommerce Checkout: Uses the standard WooCommerce cart and checkout flow
  • Quick Checkout: Plugin's built-in lightweight checkout for faster rentals
  • Redirect After Add to Cart: Choose cart page, checkout page, or stay on product page
  • Guest Checkout: Allow rentals without requiring a user account
  • Terms & Conditions: Require agreement to rental T&Cs at checkout

Pricing Settings Tab

Default Pricing

  • Default Pricing Model: Per Day, Per Hour, Per Day + Per Hour, or Flat Rate
  • Currency Format: Inherits WooCommerce currency settings; can override for rentals only
  • Price Display: Show prices with or without tax (incl./excl.)
  • Minimum Price: Set a global minimum rental price to prevent under-pricing
  • Maximum Price: Set a global cap on rental prices for listing consistency

Tax & Fees

  • Tax Class: Assign a WooCommerce tax class specifically for rental items
  • Service Fee: Add a percentage-based service fee to all rental orders
  • Deposit Requirement: Require an upfront deposit (percentage or fixed amount)
  • Late Fee Settings: Configure default late-return penalties (requires the Late Return Fees addon)

Availability Settings Tab

Business Hours

  • Operating Days: Select which days of the week your business operates
  • Pickup Time Slots: Define available pickup times (e.g., 9 AM, 10 AM, 11 AM)
  • Return Time Slots: Define available return times (e.g., 4 PM, 5 PM, 6 PM)
  • Same-Day Booking: Allow or block bookings that start on the current day
  • Advance Booking: Set a minimum and maximum number of days in advance for bookings

Off-Days & Holidays

  • Weekly Off-Days: Mark entire weekdays as non-operational (e.g., all Sundays)
  • Holiday Dates: Add specific dates when the business is closed (e.g., Christmas, New Year)
  • Recurring Holidays: Set holidays that repeat annually
  • Seasonal Closures: Block longer periods for maintenance or low season
  • Off-days can also be set per product for individual item schedules

Buffer & Preparation

  • Booking Buffer: Hours or days between consecutive bookings for cleaning/inspection
  • Preparation Time: Minimum time before a booking starts for item preparation
  • Max Booking Duration: Maximum number of days a single booking can span
  • Min Booking Duration: Minimum rental period (e.g., 2-day minimum for equipment)
  • Duration limits can be overridden per product using the Min-Max addon

Calendar Sync

  • Google Calendar: Connect your Google Calendar for two-way sync (requires the Availability Calendar Sync addon)
  • Outlook Calendar: Sync with Microsoft Outlook for corporate calendars
  • iCal Feed: Export an iCal feed URL for external calendar apps
  • Sync Interval: Set how often the calendars synchronize (every 15, 30, or 60 minutes)

Display Settings Tab

Front-End Layout

  • Default Style: Choose between Grid and List as the default shortcode style
  • Grid Columns: Default number of columns (1–6)
  • Items Per Page: Default number of items shown per page in shortcodes
  • Image Size: Set the thumbnail size for rental listing images
  • Details Page Layout: Choose tabbed or single-page layout for the single product view

Search Form

  • Search Page: Select the WordPress page where search results are displayed
  • Show Date Fields: Show or hide pickup/return date fields in the search form
  • Show Type Filter: Display the rental type dropdown in the search form
  • Show Category Filter: Display the category dropdown in the search form
  • AJAX Search: Enable live AJAX filtering without page reload

Translation & Localization

Language Files

  • The plugin is translation-ready with a .pot file included in the /languages/ folder
  • Use Loco Translate plugin to translate strings directly from the WordPress admin
  • Use WPML or Polylang for multilingual rental sites with language switchers
  • Translatable strings include: labels, buttons, error messages, and email templates
  • Custom label changes made in settings are stored as site options, not translation files

RTL Support

  • The plugin includes basic RTL (right-to-left) stylesheet support
  • Automatically loaded when WordPress is set to an RTL language (Arabic, Hebrew, Persian, etc.)
  • Override RTL styles by adding custom CSS in the WordPress Customizer
  • All shortcodes and front-end elements respect the site's text direction setting

Advanced Settings

API & Integrations

  • Google Maps API Key: Required for the search form's location picker and delivery zone mapping
  • REST API: Enable the plugin's REST API endpoints for headless or third-party integration
  • Webhooks: Configure outgoing webhooks for booking creation, status changes, and cancellations
  • Analytics Tracking: Enable Google Analytics or Meta Pixel tracking for rental conversion events

Email Configuration

  • From Name: Sender name for all rental-related emails
  • From Address: Sender email address for rental notifications
  • Email Type: Choose between HTML, Plain Text, or Multipart
  • CC & BCC: Add additional recipients for booking confirmation emails
  • Email templates can be overridden by copying them to your theme's /woocommerce/emails/ folder
