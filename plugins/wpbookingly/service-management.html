Creating Services

Services are the core offerings that customers book. Each service has its own pricing, duration, and configuration options.

Add a New Service

  • Go to Service Booking > Add New Service Booking in the admin menu
  • Enter a descriptive service title and optional subtitle
  • Write a detailed description that customers will see
  • Set a featured image and gallery images for the service

Service Details

  • Configure the service price as a fixed or custom rate
  • Set the service duration in minutes or hours
  • Choose between one-time and recurring booking options
  • Enable buffer time between consecutive appointments

Pricing Options

WpBookingly offers flexible pricing models to accommodate different business types and service structures.

Fixed Pricing

  • Set a single flat rate for the entire service
  • Ideal for standard offerings with no variations
  • Works with WooCommerce for tax and discount calculations
  • Supports free consultations by setting price to $0

Custom & Variable Pricing

  • Set different prices based on staff member or location
  • Create tiered pricing for group bookings
  • Add extra service options with individual price tags
  • Use the Service Package Bundler addon for package deals

Service Categories & Organization

Organize your services into logical categories to help customers find what they need quickly.

Creating Categories

  • Navigate to Service Booking > Service Categories
  • Add category names, descriptions, and images
  • Organize categories hierarchically for sub-groupings
  • Assign services to one or more categories

Category Display

  • Categories appear as filterable tabs on the booking form
  • Use shortcode parameters to show specific categories
  • Reorder categories with drag-and-drop interface
  • Set a default category that opens on page load

Extra Services & Addons

Enhance primary services with optional extras that customers can add during the booking process.

Extra Service Items

  • Add optional extras like additional cleaning, express delivery, etc.
  • Each extra has its own name, description, and price
  • Extras can be mandatory or optional selection
  • Set maximum quantity limits per extra item

Service Package Bundler

  • Combine multiple services into discounted packages
  • Requires the Service Package Bundler addon
  • Set package validity periods and usage limits
  • Offer bundle-only services not available individually

Service Visibility & Availability

Control when and where services appear to customers to manage your booking workflow effectively.

Visibility Settings

  • Publish or unpublish services individually
  • Schedule services to become available on specific dates
  • Hide services from unregistered users
  • Set service-specific booking cutoff times

Location Management

  • Assign services to specific physical locations
  • Configure different availability per location
  • Display location maps and addresses on booking forms
  • Staff can be assigned per-service and per-location
