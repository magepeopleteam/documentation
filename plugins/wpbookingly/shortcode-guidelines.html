Primary Booking Shortcodes

Shortcodes allow you to embed booking forms and service displays anywhere on your WordPress site.

Main Booking Form

  • [wpbookingly_booking] renders the full multi-step booking form
  • Displays service selection, date picker, time slots, and checkout
  • Automatically adapts to the configured booking flow
  • Best placed on a dedicated booking page

Service List

  • [wpbookingly_services] displays a browsable service catalog
  • Shows service name, description, price, and duration
  • Supports category filtering and search
  • Clicking a service redirects to the booking form

Staff Availability

  • [wpbookingly_staff] shows staff profiles and availability
  • Displays staff name, photo, bio, and assigned services
  • Shows real-time availability for the current week
  • Supports direct booking via staff selection

Category Grid

  • [wpbookingly_categories] renders a visual category grid
  • Each category card shows name, image, and service count
  • Clicking a category filters the service list
  • Customize grid columns via shortcode parameters

Shortcode Parameters

Customize the behavior and appearance of shortcodes using optional parameters.

Layout & Styling

  • layout — Choose inline, horizontal, or vertical form layout
  • columns — Set number of grid columns (1-4, default 2)
  • show_search — Enable or disable the search bar (true/false)
  • show_filter — Show or hide category filter tabs (true/false)

Content Filtering

  • category — Display services from a specific category slug
  • staff_id — Show availability for a specific staff member
  • service_id — Pre-select a specific service on load
  • limit — Maximum number of items to display

Booking Flow

  • show_progress — Enable multi-step progress indicator
  • show_summary — Display booking summary before checkout
  • guest_checkout — Allow booking without registration
  • redirect — Custom URL after successful booking

Date & Time

  • date_format — Override the default date display format
  • time_format — Override the default time display format
  • first_day — Set the starting day of the week (0=Sun, 1=Mon)
  • min_date — Earliest selectable date for booking

Usage Examples

Practical examples showing how to use shortcodes in different scenarios.

Basic Booking Form

  • [wpbookingly_booking] — Full booking experience
  • [wpbookingly_booking layout="horizontal"] — Horizontal layout
  • [wpbookingly_booking show_progress="true"] — With progress bar
  • [wpbookingly_booking guest_checkout="true"] — Guest only

Filtered Displays

  • [wpbookingly_services category="consultation"] — Single category
  • [wpbookingly_services columns="3" show_search="true"] — Grid with search
  • [wpbookingly_staff staff_id="5"] — Specific staff availability
  • [wpbookingly_categories columns="4"] — 4-column category grid

Embedding & Placement Tips

Best practices for placing shortcodes on your WordPress site for optimal user experience.

Page Placement

  • Place the main booking shortcode on a dedicated, uncluttered page
  • Use the service list shortcode on category landing pages
  • Embed staff availability on team or about pages
  • Avoid placing multiple booking forms on the same page

Widget Areas

  • Add the booking form to a sidebar using a WordPress text widget
  • Use Gutenberg shortcode blocks for easy placement in FSE themes
  • Elementor users can use the Shortcode widget
  • Test mobile rendering after embedding in widget areas
