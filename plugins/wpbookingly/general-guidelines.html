Customization Options

Tailor WpBookingly to match your brand and business requirements through extensive customization settings.

Branding & Labels

  • Change the "Service Booking" label and slug throughout the site
  • Customize button text, form labels, and confirmation messages
  • Upload a custom logo for the booking form header
  • Modify colors and typography via CSS hooks

Form Customization

  • Rearrange form field order via drag-and-drop
  • Add or remove customer information fields
  • Make specific fields required or optional
  • Enable the attendee registration form for group bookings

Translations & Multilingual Support

WpBookingly is fully translatable and compatible with popular multilingual plugins.

Translation Methods

  • Use Loco Translate for in-browser .po/.mo file editing
  • Translate via WPML for full multilingual site integration
  • Edit language files directly in /wp-content/languages/plugins/
  • Submit translations to WordPress.org for community contribution

Multilingual Strategy

  • Translate service names, descriptions, and categories per language
  • Set language-specific booking form labels and messages
  • Configure separate booking pages for each language
  • Currency switching via WPML + WooCommerce multilingual

Email Notification System

Automated email notifications keep customers and staff informed at every stage of the booking lifecycle.

Notification Types

  • New Booking — Sent when a customer places a booking
  • Confirmation — Sent when booking status changes to confirmed
  • Reminder — Sent before the appointment (customizable timing)
  • Cancellation — Sent when a booking is cancelled
  • Reschedule — Sent when a booking date/time changes

Template Customization

  • Edit email templates from WpBookingly > Email Settings
  • Use placeholders for dynamic content (customer name, service, date)
  • Customize sender name and from email address
  • Send test emails to verify template rendering

Recipient Management

  • Send notifications to the booking customer
  • Send copies to assigned staff members
  • Configure admin email addresses for all notifications
  • Set different notification rules per service

SMS Notifications

  • Requires the SMS and Email Reminder addon
  • Send SMS reminders for upcoming appointments
  • Enable two-way SMS conversation for rescheduling
  • Configure SMS templates separately from email

Customer Experience

Deliver a seamless booking experience with features designed for customer convenience.

Booking Flow

  • Multi-step booking wizard guides customers through selection
  • Progress indicator shows booking stage completion
  • Real-time availability checking prevents double bookings
  • Mobile-responsive forms work on all device sizes

Self-Service Portal

  • My Account integration for booking history and management
  • Email links provide one-click access to modify bookings
  • Customers can cancel or reschedule within your policy window
  • Download invoices and booking receipts

Video Conferencing

  • Integrate with Zoom, Google Meet, or other video platforms
  • Automatically generate meeting links for virtual consultations
  • Include meeting details in confirmation emails
  • Staff can create recurring meeting links

Customer Reviews

  • Requires the Customer Review Collector addon
  • Send automated review requests after completed bookings
  • Display ratings and reviews on service pages
  • Moderate reviews before publication

Performance & Security

Best practices for keeping your booking system fast, secure, and reliable.

Caching & Performance

  • Use a caching plugin for public pages (W3 Total Cache, WP Rocket)
  • Exclude booking pages from cache to ensure real-time availability
  • Enable database optimization for booking tables
  • Use a CDN for static assets

Security Measures

  • Keep WpBookingly and addons updated to the latest version
  • Use reCAPTCHA on booking forms to prevent spam
  • Enable SSL certificate for secure payment transactions
  • Regularly back up your database and files
