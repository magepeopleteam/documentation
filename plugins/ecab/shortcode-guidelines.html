Shortcode Guidelines

Display transport booking forms anywhere on your site using dedicated shortcodes. eCab provides separate shortcodes for each pricing model, giving you full control over the customer booking experience.

Distance-Based Pricing Shortcode

Use this shortcode to display a booking form that calculates fares based on pickup-to-dropoff distance. Customers select their pickup and dropoff locations, and the fare is computed per kilometer plus per-hour waiting charges.

  • Shortcode automatically detects the distance-based pricing model
  • Google Maps integration provides address autocomplete for pickup and dropoff
  • Fare is dynamically calculated based on the route distance and duration
  • Supports additional parameters like vehicle type and passenger count filtering
  • Real-time fare preview updates as the customer adjusts pickup and dropoff locations

Fixed Hourly Pricing Shortcode

Display a booking form tailored for hourly rental services. Customers book a vehicle for a set number of hours with a flat hourly rate, ideal for airport transfers, city tours, or event transportation.

  • Customers select pickup location and duration in hours
  • Fare is calculated as hourly rate multiplied by the number of hours
  • Supports minimum hour requirements (e.g., minimum 3 hours per booking)
  • Can include extra charges for additional distance beyond included mileage
  • Waiting time and overtime charges can be configured separately

Manual Fixed Location Pricing Shortcode

Use this shortcode when you have predefined routes with fixed prices. Customers select from a list of pickup and dropoff location pairs, each with a predetermined fare, providing a simple and predictable booking experience.

  • Customers choose from predefined location pairs (e.g., Airport to Downtown Hotel)
  • Each location pair has a manually configured fixed price
  • Perfect for standard routes with known distances and traffic patterns
  • Dropdown-style selection simplifies the booking process for customers
  • Can include multiple vehicle types with different fixed prices per route

Embedding Shortcodes

  • Place the shortcode in any WordPress post, page, or custom post type
  • Use the Classic Editor by pasting the shortcode directly into the editor
  • In Gutenberg, add a Shortcode block and paste the shortcode code
  • In Elementor, use the Shortcode widget to embed eCab booking forms
  • Shortcodes can also be added to widget areas, footers, or sidebars
  • Wrap shortcodes in custom HTML containers for additional styling control

Shortcode Parameters

  • style: control the layout style of the booking form (horizontal, vertical)
  • class: add custom CSS classes to the shortcode wrapper for branding
  • vehicle_id: pre-select a specific transport vehicle by its ID
  • hide_location: set to yes to hide the location selection step
  • hide_date: set to yes to hide the date/time selection step
  • hide_passenger: set to yes to hide the passenger/bag count selector

Troubleshooting Shortcodes

  • If the shortcode displays as plain text, ensure permalinks are saved (Settings → Permalinks → Save)
  • Verify WooCommerce is installed and activated for payment processing
  • Check that the selected pricing model matches the shortcode type
  • Clear any caching plugins or CDN caches to reload the shortcode output
  • Test the shortcode on a default WordPress theme to rule out theme conflicts
  • Enable WordPress debugging to identify PHP errors from conflicting plugins
