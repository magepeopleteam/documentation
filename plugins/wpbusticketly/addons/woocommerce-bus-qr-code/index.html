QR Code — WpBusTicketly Add‑On

Add QR code support to your bus ticket booking system for fast and secure ticket validation.

Overview

Add QR code support to your bus ticket booking system for fast and secure ticket validation. This is a PRO feature of the WpBusTicketly plugin.

Settings

To enable QR code, go to Bus → Settings → QR Settings:

  • Enable/Disable — Toggle the QR code feature on or off.
  • Checker Role — Set the user role for ticket checkers who can scan and validate tickets.

Usage

Once enabled, QR codes appear on all PDF tickets. The checker scans the ticket to validate it. Check-in status is updated in real time and can be viewed in the Passenger List.
