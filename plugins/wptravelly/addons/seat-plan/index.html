Seat Plan Addon

Give your travelers control over seat selection during booking. Improves the booking experience and ensures every customer can pick their preferred seats easily.

Installation

  1. Download the addon zip from your account.
  2. Go to Plugins → Add New → Upload Plugin.
  3. Upload the zip and click Install Now.
  4. Activate the plugin from the Plugins page.

Usage

  1. Go to the event edit page and enable the Seat Plan option.
  2. Configure the seat layout using the visual seat map editor.
  3. Assign seat types and pricing for different sections.
  4. Customers can now select their preferred seats during booking.
