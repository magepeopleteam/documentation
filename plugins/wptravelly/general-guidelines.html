General Guidelines

Manage tour taxonomies, create packages, configure booking settings, and follow best practices for a smooth tour booking experience with WpTravelly.

Tour Management

Creating a New Tour

  • Go to WpTravelly → Add New Tour in the WordPress admin
  • Enter the tour title, description, and featured image
  • Set tour type: single-day, multi-day, or recurring
  • Define pricing: regular price, sale price, and group rates
  • Add itinerary sections with day-wise descriptions

Managing Tour Categories

  • Navigate to WpTravelly → Categories to add/edit categories
  • Assign a name, slug, description, and featured image
  • Organise categories hierarchically if needed (parent/child)
  • Use categories to group tours by theme or difficulty level
  • Display categories using [wptravelly-category-list]

Tour Organizers

  • Add tour organizers under WpTravelly → Organizers
  • Include name, bio, photo, contact email, and phone
  • Assign social media links (Facebook, Instagram, Twitter)
  • Link organizers to tours via the tour edit screen
  • Display organizers using [wptravelly-organizer-list]

Tour Tags & Activities

  • Create tags for quick filtering (e.g. "Beach", "Hiking")
  • Define activities to highlight tour experiences
  • Tags and activities appear as filter options in front-end
  • Display tag lists with [wptravelly-tag-list]
  • Display activity lists with [wptravelly-activity-list]

Booking & Pricing Configuration

Pricing Options

  • Set base price per adult with optional child/infant pricing
  • Configure group discounts for large parties
  • Enable seasonal pricing for peak and off-peak periods
  • Offer deposit or partial payment options
  • Define currency symbol and position under settings

Booking Rules

  • Set minimum and maximum travellers per booking
  • Define booking cutoff time (e.g. 24 hours before departure)
  • Enable automatic or manual booking confirmation
  • Configure cancellation policy and refund rules
  • Set buffer time between consecutive tour slots

Calendar & Availability

  • Define available dates and time slots per tour
  • Set recurring schedules (daily, weekly, monthly)
  • Block out dates manually or via availability rules
  • Display real-time availability on the front end
  • Prevent overbooking by setting capacity limits

Email Notifications

  • Customise admin notification for new bookings
  • Configure customer confirmation and reminder emails
  • Set up cancellation and refund notification templates
  • Use shortcodes in email templates for dynamic data
  • Test email delivery with a sandbox booking

Display & Layout Tips

Shortcode Placement

  • Use [wptravelly-tour-list column=3] on the main tours page
  • Combine [wptravelly-tour-search] with [wptravelly-tour-list-filter]
  • Place [wptravelly-top-attraction] on the homepage
  • Add [wptravelly-travel-location] on destination pages
  • Avoid stacking multiple heavy shortcodes on a single page

Style Selection Guide

  • List — compact, text-heavy, good for directories
  • Grid — balanced, image+text, most versatile
  • Modern — clean cards, ideal for professional agencies
  • Blossom / Orchid / Lotus / Flora — visual themes for branding
  • Test each style with your content to choose the best fit

Content Best Practices

  • Write compelling tour descriptions with clear highlights
  • Use high-quality images (1200×800 px or larger)
  • Keep tour titles concise and keyword-rich
  • Include pricing prominently in tour cards
  • Add customer reviews and ratings to build trust

SEO Optimisation

  • Set unique meta titles and descriptions per tour
  • Use SEO-friendly URL slugs for tours and taxonomy pages
  • Enable breadcrumbs for better site navigation
  • Add schema markup for tour listings (JSON-LD)
  • Submit tour sitemaps to Google Search Console
