Shortcode Guidelines

WpTravelly provides a rich set of shortcodes to display tours in list, grid, modern, blossom, orchid, lotus, and flora styles with filtering, pagination, and sorting options.

Tour Display Shortcodes

Tour List (Default)

  • [wptravelly-tour-list column=3] — default list view
  • [wptravelly-tour-list type='trending' column=4] — trending tours
  • [wptravelly-tour-list type='popular' column=3] — popular tours
  • [wptravelly-tour-list type='featured' column=4] — featured tours
  • Supports attributes: column, type, style, category

Modern List View

  • [wptravelly-modern-list column=3] — modern card layout
  • [wptravelly-modern-list column=2 style='box'] — box style modern
  • [wptravelly-modern-list category='adventure'] — filtered by category
  • Ideal for travel agencies showcasing curated packages
  • Works with all type filter values

Grid View (Flora / Blossom / Orchid)

  • [wptravelly-flora-grid column=3] — flora card design
  • [wptravelly-blossom-grid column=4] — blossom style grid
  • [wptravelly-orchid-grid column=3] — orchid elegant layout
  • Each style provides unique Card UI and hover effects
  • Use column=2/3/4 to control columns

Lotus Grid View

  • [wptravelly-lotus-grid column=3] — lotus theme grid
  • Designed for premium / luxury tour presentation
  • Displays price, duration, and rating per card
  • Supports category and organizer filtering
  • Fully responsive across all devices

Specialized Shortcodes

Top Attractions

  • [wptravelly-top-attraction column=3]
  • [wptravelly-top-attraction column=4 type='popular']
  • Highlights top-rated or most-visited tours
  • Supports the slider attribute for carousel display

Travel Location

  • [wptravelly-travel-location column=4]
  • [wptravelly-travel-location style='list']
  • Displays tours grouped by destination or location
  • Useful for "Tours in {City}" style sections

Tour List with Top Filter

  • [wptravelly-tour-list-filter column=3]
  • Adds a filter bar above the tour grid
  • Users can filter by category, price range, duration
  • Ideal for comprehensive tour directories

Search Shortcode

  • [wptravelly-tour-search] — full search form
  • [wptravelly-tour-search style='horizontal'] — inline layout
  • Search by keyword, destination, date, or category
  • Results update dynamically (AJAX-based)

Taxonomy Shortcodes

Category List

  • [wptravelly-category-list column=4]
  • [wptravelly-category-list style='grid']
  • Displays all tour categories with linked archive pages
  • Each category shows name, image, and tour count

Organizer List

  • [wptravelly-organizer-list column=3]
  • Showcases tour guides, agencies, or operators
  • Cards include photo, bio snippet, and contact info
  • Clicking navigates to the organizer archive

Tag & Activity List

  • [wptravelly-tag-list column=4]
  • [wptravelly-activity-list column=3]
  • Displays tag cloud or activity icons
  • Useful for building filter or browse-by-interest pages

Common Parameters

  • column — number of columns (2, 3, or 4)
  • type — all, trending, popular, featured
  • style — grid, list, box, modern
  • category — slug of the tour category
  • organizer — slug of the tour organizer
