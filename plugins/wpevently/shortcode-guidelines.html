Events List Style Shortcode

Display events in a vertical list layout with event thumbnails, dates, titles, and ticket buttons. Ideal for sidebars, agendas, and schedule pages.

Basic Usage

  • [mep_event_list] — Shows all upcoming events in list format
  • [mep_event_list limit="6"] — Limits to 6 events
  • [mep_event_list show="3"] — Shows 3 events per page with pagination
  • [mep_event_list category="conference"] — Filters by event category slug
  • [mep_event_list order="asc"] — Sorts events in ascending date order

Parameters

  • limit — Number of events to display (default: all)
  • show — Events per page for paginated lists
  • category — Filter by event category slug
  • order — Sort order: asc or desc
  • style — Layout style: list or grid

Events Grid Style Shortcode

Display events in a responsive card grid. Each card shows the event thumbnail, date, title, location, and a book now button.

Basic Usage

  • [mep_event_grid] — Shows all upcoming events in grid layout
  • [mep_event_grid column="3"] — Sets grid to 3 columns
  • [mep_event_grid column="4" limit="8"] — 4 columns showing 8 events
  • [mep_event_grid category="workshop"] — Filters by category slug
  • [mep_event_grid show_footer="no"] — Hides the footer area

Parameters

  • column — Number of grid columns (2, 3, or 4, default: 3)
  • limit — Number of events to display
  • category — Filter by event category slug
  • show_footer — Show/hide the card footer (yes/no)
  • order — Sort order: asc or desc

Parameters

  • limit — Number of events to display
  • category — Filter by event category slug
  • show_past — Include past events (yes/no)
  • order — Sort order: asc or desc
  • Ideal for sidebar widgets and footer areas

Events Native Style Shortcode

Renders events using the native WordPress loop styling. Events blend seamlessly with your theme's default post layout for a consistent look.

Basic Usage

  • [mep_event_native] — Displays events using native theme styling
  • [mep_event_native limit="4"] — Shows 4 events
  • [mep_event_native category="meetup"] — Filter by category
  • [mep_event_native show="3"] — Pagination with 3 events per page
  • Respects your theme's archive and single post templates

Parameters

  • limit — Total events to display
  • show — Events per page with pagination
  • category — Filter by event category slug
  • order — Sort order: asc or desc
  • Use this shortcode for archive-like event pages

Events Calendar Shortcode

Display a full interactive calendar with events marked on their respective dates. Users can navigate between months and click events for details.

Basic Usage

  • [mep_calendar] — Full calendar with all upcoming events
  • [mep_calendar category="conference"] — Calendar filtered by category
  • [mep_calendar view="month"] — Default month view
  • [mep_calendar view="week"] — Week-by-week view
  • Clicking a date shows events scheduled for that day

Parameters

  • category — Filter events displayed on the calendar
  • view — Initial calendar view: month, week, or day
  • show_past — Display past events on the calendar
  • AJAX-powered navigation between months and weeks
  • Responsive design works on mobile devices

Recurring Events Shortcode

Display recurring event series with all future occurrences. Useful for weekly classes, monthly meetups, and multi-date conferences.

Basic Usage

  • [mep_recurring_event] — Shows all recurring event series
  • [mep_recurring_event limit="3"] — Limits to 3 series
  • [mep_recurring_event category="class"] — Filter by category
  • Each entry shows the next upcoming date in the series
  • Click through to view all dates for a specific series

Parameters

  • limit — Number of recurring series to display
  • category — Filter recurring events by category slug
  • order — Sort order: asc or desc
  • Displays the next occurrence date for each series
  • Works with daily, weekly, monthly, and custom intervals

Events Carousel Style Shortcode

Display events in a horizontal carousel slider with navigation arrows and optional autoplay. Great for hero sections, homepage banners, and featured content areas.

Basic Usage

  • [mep_event_carousel] — Events carousel with default settings
  • [mep_event_carousel limit="8"] — Shows 8 events in the carousel
  • [mep_event_carousel slide="3"] — 3 events visible per slide
  • [mep_event_carousel autoplay="yes"] — Enables autoplay
  • [mep_event_carousel category="featured"] — Filter by category

Parameters

  • limit — Total events in the carousel
  • slide — Number of events visible per slide (1-4)
  • autoplay — Enable automatic sliding (yes/no)
  • category — Filter by event category slug
  • loop — Infinite loop navigation (yes/no)

Events by Category Shortcode

Display events filtered by a single category. You can specify the category slug and choose between list or grid layout for the output.

Basic Usage

  • [mep_event_category category="workshop"] — Events in Workshop category
  • [mep_event_category category="conference" layout="grid"] — Grid layout
  • [mep_event_category category="seminar" limit="4"] — Limit to 4 events
  • [mep_event_category category="meetup" show="2"] — 2 events per page
  • Replace workshop with your actual category slug

Parameters

  • category — (Required) The event category slug to filter by
  • layout — Display layout: list or grid
  • limit — Number of events to display
  • show — Events per page for paginated views
  • order — Sort order: asc or desc

Common Shortcode Parameters

These parameters are shared across multiple shortcodes and control global display behaviour such as date formats, filters, and styling.

Date & Filtering

  • from_date — Show events starting from this date (YYYY-MM-DD)
  • to_date — Show events up to this date (YYYY-MM-DD)
  • show_past — Include past events in results
  • hide_expired — Automatically hide expired events
  • city — Filter events by city name

Display Controls

  • column — Number of grid columns (2-4)
  • order — Sort direction: asc or desc
  • show_footer — Show/hide card footer elements
  • pagination — Enable or disable pagination
  • class — Additional CSS class for custom styling
