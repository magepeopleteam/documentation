WooCommerce Event Coupon Code Addon

Create and manage discount coupons for your events. Offer percentage-based or fixed-price discounts, set usage limits, and restrict coupons to specific events or ticket types.

Features

  • Percentage and fixed amount discount types
  • Per-coupon usage limits (global and per-user)
  • Minimum spend and maximum spend restrictions
  • Event-specific coupon assignment
  • Expiry date and time for time-limited promotions
  • Exclude sale items and specific ticket types

Configuration

  • Install the Coupon Code addon and activate your license
  • Go to WooCommerce → Coupons to create new coupons
  • Set discount type, amount, and usage restrictions
  • Assign the coupon to specific events in the coupon data meta box
  • Customers enter the coupon code at checkout to receive the discount

WooCommerce Event Waitlist Addon

Allow attendees to join a waitlist when an event sells out. Waitlisted users are automatically notified when tickets become available, ensuring no booking opportunity is missed.

Features

  • Automatic waitlist display when tickets are sold out
  • Email notifications when tickets become available
  • First-come, first-served queue system
  • Configurable maximum waitlist size
  • Admin overview of waitlist status per event
  • Automatic expiration of stale waitlist entries

Usage

  • Activate the addon and configure global settings
  • Enable or disable waitlist per individual event
  • Customers click "Join Waitlist" on the sold-out event page
  • When a ticket is freed (cancellation), the next person is notified
  • Track waitlist size and conversion rates in reports

Event Max-Min Quantity Limiting Addon

Control the minimum and maximum number of tickets a customer can purchase in a single transaction. Useful for limiting group sizes or ensuring minimum order values.

Features

  • Set minimum ticket quantity per order
  • Set maximum ticket quantity per order
  • Apply limits globally or per event
  • Apply limits per ticket type
  • User-friendly validation messages at checkout
  • Compatible with all ticket types including variable pricing

Use Cases

  • Minimum 2 tickets per order for group events
  • Maximum 10 tickets per customer to prevent bulk buying
  • Limit VIP ticket purchases to 2 per person
  • Enforce minimum spend requirements for events
  • Prevent scalping with strict per-customer limits

Membership Price Addon

Offer discounted ticket prices to members of your site. Integrates with popular membership plugins to provide exclusive pricing for registered members.

Features

  • Member-only ticket pricing and discounts
  • Integration with WooCommerce Memberships and other membership plugins
  • Percentage or fixed discount for members
  • Different pricing tiers for different membership levels
  • Member prices displayed prominently on event pages
  • Non-members see regular prices with upgrade prompts

Setup

  • Install and activate the Membership Price addon
  • Configure member discount rules in addon settings
  • Set default member discount percentage
  • Override discount for individual events as needed
  • Test with different membership levels to verify pricing

WooCommerce Event Calendar Addon

An enhanced frontend calendar with additional views, filtering options, and styling controls. Provides a richer calendar experience than the built-in shortcode.

Features

  • Month, week, and day view options
  • Category and tag filtering on the calendar
  • Event tooltips on hover with key details
  • Click-to-view event details in a modal or new page
  • AJAX navigation between months without page reload
  • Custom colour coding by event category

Configuration

  • Activate the Calendar addon and configure display settings
  • Use shortcode [mep_calendar_pro] for the enhanced calendar
  • Set default view, first day of week, and time format
  • Enable or disable event tooltips and modal popups
  • Assign category colours in the addon settings panel

WooCommerce Event QR Code Addon

Generate unique QR codes on every ticket for secure attendee check-in. QR codes can be scanned from printed tickets or mobile devices for fast entry management.

Features

  • Unique QR code per ticket
  • Customisable QR code size and position on PDF tickets
  • QR code on both PDF and email tickets
  • One-time scan validation prevents duplicate entry
  • Check-in status tracking in the attendee list
  • Configurable error correction level for reliable scanning

Scanning Workflow

  • Attendee receives ticket with QR code via email
  • At the venue, staff scans the QR code using any scanner app
  • The scan validates the ticket against the database
  • Attendee is marked as checked in — status reflected in admin
  • Scanned tickets cannot be reused, preventing fraud

Seat Plan Addon for Event Manager

Create interactive seating charts for events with assigned seating. Attendees can select their preferred seats from a visual map during checkout.

Features

  • Visual, drag-and-drop seat map editor
  • Define sections, rows, and individual seats
  • Different pricing per section or seat
  • Real-time seat availability updates
  • Reserved and wheelchair-accessible seat markers
  • Mobile-responsive seat selection interface

Setup

  • Activate the Seat Plan addon and create a venue layout
  • Add sections (e.g., VIP, General, Balcony) with seat counts
  • Assign pricing per section or individual seats
  • Attach the seat plan to an event in the event edit screen
  • Attendees select their seats from the interactive map at checkout

Event Marketplace / Frontend Submit Addon

Allow users to submit and manage their own events from the frontend of your site. Turn your site into a community-driven event marketplace.

Features

  • Frontend event submission form with custom fields
  • User dashboard to manage submitted events
  • Pending review workflow — admin approval required
  • Commission or flat-fee pricing for event submissions
  • User profile pages showing their events
  • Email notifications for submission status changes

Configuration

  • Activate the Marketplace addon and configure submission settings
  • Create the frontend submission form with required fields
  • Set pricing for event submissions (free, paid, commission)
  • Configure moderation settings for new submissions
  • Use shortcode [mep_event_submission] on any page

Installing Addons

All WpEvently addons are installed via zip upload or automatic installation, just like the main plugin. Follow these steps for a smooth addon setup.

Installation Steps

  • Download the addon .zip file from your Magepeople account
  • Go to Plugins → Add New → Upload Plugin
  • Select the addon .zip file and click Install Now
  • Activate the addon after installation completes
  • Enter your license key if prompted for premium addons

Compatibility

  • All addons require the base WpEvently plugin to be active
  • Most addons are compatible with both free and Pro versions
  • Check the addon documentation for specific version requirements
  • Test addons on a staging site before production deployment
  • Keep all addons updated to the latest versions
