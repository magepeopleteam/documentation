How to Override and Change Event Templates

Sometimes you may need to change some of the default design or content to your own design, such as if you want to change the icons or add/remove a new class. We have the override feature in the WooCommerce Event Manager plugin. So you can do it easily and very smartly.

For example, you want to change the default icons of the sidebar address list. To do this please follow the process below:

  1. First, go to your theme folder and create a new folder named mage-events. Then enter into this newly created folder. Create another folder into it name it as single.
  2. Now go to the plugins/mage-eventpress/templates folder. Copy the location_list.php file.
  3. Go back to your theme folder then go to mage-events/single folder and paste the template file.
  4. Now open the template file and change the icons. The plugin always loads this template file location_list.php from your theme folder. So this change will never be lost in any future update of the plugin.

You can follow this process to override any template files of the WooCommerce Event Manager Plugin.

How to Create New Theme/Template

WooCommerce Events Manager plugin is fully developer friendly. It comes with 4 pre-active themes, but most of the time clients want to make a custom theme which is similar to their theme or any design. That’s why we made our plugin fully flexible so you can easily create a new theme/template or override any default theme. If you have experience in customizing WooCommerce template then it will be very easy for you. It’s like WooCommerce templating.

  1. First, you need to copy the mage-eventpress/templates/ folder from the plugin folder, and paste it into your theme folder and rename it as mage-events: theme-folder/mage-events/
  2. Now you can create as many templates/themes as you want. You just need to put your theme files into: theme-folder/mage-events/themes/
  3. Suppose you want to make a theme and the file name will be dhaka.php. The file structure will be: theme-folder/mage-events/themes/dhaka.php
  4. Then you need to create a theme name, just like WordPress page template naming. You just need to set the name in the first line by PHP comment: // Template Name: Dhaka

That’s it! You can now find this theme in the template list which you can find in the event page in dashboard and the event settings page.

How to Create Your Own PDF Ticket Design

We have a PDF Ticket add-on that generates a PDF Ticket for a registered attendee of an event. By default, there is a simple basic PDF ticket design. But if you want you can create your own design PDF ticket.

  1. First copy this folder from the PDF Addon Plugin: woocommerce-event-manager-pdf-ticket/templates/pdf-theme
  2. Then paste it into your theme folder, and rename the pdf-theme folder to mage-events-pdf-themes
  3. Now create your own theme. Create a new PHP file name it as you want. Then open the file and add this line at the beginning of the file: <?php // Template Name: YOUR THEME NAME HERE
  4. Now start your Ticket design HTML. You can code as you wish, but remember there are some limitations for CSS attributes for PDF generation.

How to Override and Change/Add New Social Share Icon

The WooCommerce Event Manager plugin has a wonderful template override feature. You can modify or add any content to any template. If you want to change/add a new social share icon in the Share button template, please follow the process below.

  1. First, go to your theme folder and create a new folder named mage-events. Then enter into this newly created folder. Create another folder into it name it a single.
  2. Now go to the plugins/mage-eventpress/templates folder. Copy the share_btn.php file.
  3. Go back to your theme folder then go to mage-events/single folder and paste the template file.
  4. Now you can edit this file as you wish. You can add a new share button or change the existing button.

How to Override and Change Event Category or Organizer Page Templates

The WooCommerce Events Manager plugin is fully developer-friendly. You can follow the steps below if you want to change or override any part of the category or organizer list page design. If you have experience customizing WooCommerce templates, it will be effortless for you. It’s like WooCommerce templating.

  1. First, you need to create a new folder in your active theme folder and rename it as mage-events: theme-folder/mage-events/
  2. Now, please go to wp-content/plugins/mage-eventpress/templates folder and copy the file taxonomy-category.php or the taxonomy-organizer.php file and paste it to the theme-folder/mage-events/ folder.
  3. You can change any part of the code in the template files. Suppose you want to change the grid style to the list style of the event list on the category page. You need to change the code line 17:

Change from:

do_action('mep_event_list_shortcode', get_the_id(), 'three_column', 'grid');

Change to:

do_action('mep_event_list_shortcode', get_the_id(), 'one_column', 'list');
