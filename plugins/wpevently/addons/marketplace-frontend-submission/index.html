Marketplace / Frontend Submission Addon

This is event marketplace addon that is needed to allow event submit from frontend. Multiple organisers can sell tickets together in a same site.

Installation

  1. Download the addon zip from your account.
  2. Go to Plugins → Add New → Upload Plugin.
  3. Upload the zip and click Install Now.
  4. Activate the plugin from the Plugins page.

Shortcode

Use the following shortcode for organizer registration:

[mefs_custom_registration]

This shortcode displays the registration form that allows new organizers to sign up and start submitting events from the frontend.
