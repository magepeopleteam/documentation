Overview

With the Event Marketplace / Frontend Submit Addon, organizers can submit events from their My Account or organizer page. Organizers can also view their event ticket selling balance, see the attendee list, and download the attendee list.

Shortcode

Use the following shortcode for organizer registration:

[mefs_custom_registration]

This shortcode displays the registration form that allows new organizers to sign up and start submitting events from the frontend.
