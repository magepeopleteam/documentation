Event Max‑Min Quantity Limiting — WpEvently Add‑On

The event max-min addon is necessary for limiting the amount of tickets sold. Organizers can set a maximum buying quantity or minimum purchase quantity.

Overview

The Max-Min Quantity Limiting Addon allows the admin to enable events to sell with limited quantities per order. You can set both minimum and maximum ticket quantities that a customer can purchase in a single transaction.

Settings

After installation, navigate to the event edit page to configure quantity limits. You can set separate minimum and maximum values for each ticket type.

Frontend Display

When a customer tries to add tickets outside the allowed range, the addon displays a notification message indicating the quantity limits.

Purchase

This addon is available for purchase at the MagePeople website.
