WooCommerce Event QR Code — WpEvently Add‑On

QR code addon is needed to validate the ticket. If you have a large event then this addon must need to check ticket validity by scanning by QR scanner or mobile.

Overview

The WooCommerce Event QR Code Addon provides a simple solution for managing QR codes for attendee sign-in or check-in at the event spot. QR codes are added to PDF tickets for easy scanning at the event gate.

Settings

After installation, navigate to the addon settings to configure:

  • Enable QR Code — Toggle QR code generation on or off.
  • Checker Role — Assign which user roles (e.g., checker, admin) are allowed to scan and validate tickets.

Usage

Once enabled, QR codes appear on all PDF tickets. The checker scans the ticket using a QR scanning app:

  • Valid ticket: Shows a check-in button that has not yet been used.
  • Invalid ticket: No check-in button appears — the ticket has already been scanned or used.

Admin can see how many attendees passed through the event gate by checking the attendee list check-in status.
