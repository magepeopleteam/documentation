WooCommerce Event Coupon Code Addon

Event coupon addon that is used for marketing purpose and giving discount to customer for boosting sell.

Installation

  1. Download the addon zip from your account.
  2. Go to Plugins → Add New → Upload Plugin.
  3. Upload the zip and click Install Now.
  4. Activate the plugin from the Plugins page.

Features

  • Event-Specific Coupon — Restrict coupons to specific events only.
  • Coupon Restricted by Specific Users — Limit coupon usage to selected customers.
  • Coupon Restricted by Specific Dates — Set date ranges for coupon validity.

Usage

  1. Create a standard WooCommerce coupon.
  2. Use the addon's settings to restrict the coupon to a specific event.
  3. Coupons will only apply to the selected event's tickets during checkout.
