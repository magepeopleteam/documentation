WooCommerce Event Coupon Code — WpEvently Add‑On

Event coupon addon that is used for marketing purpose and giving discount to customer for boosting sell.

Overview

The WooCommerce Event Coupon Code Addon provides a simple solution for managing coupon codes for specific events to specific customers. It integrates with the existing WooCommerce coupon system.

Features

  • Event-Specific Coupon — Restrict coupons to specific events only.
  • Coupon Restricted by Specific Users — Limit coupon usage to selected customers.
  • Coupon Restricted by Specific Dates — Set date ranges for coupon validity.

Usage

Create a standard WooCommerce coupon and use the addon's settings to restrict it to a specific event. Coupons will only apply to the selected event's tickets during checkout.
