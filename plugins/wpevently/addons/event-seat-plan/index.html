Event Seat Plan — WpEvently Add‑On

A seat plan addon is needed for that organiser who wants to display a seat plan for customers to choose seating during event ticket buying.

Overview

This Event Plugin addon for Seat Plan allows anyone to sell tickets with a seat plan. Different seat types can have different pricing, and the system is very flexible for seat mapping.

Seat Plan Design

You can design a seat plan with a configurable number of columns and rows. Any row can be left blank, kept empty, or have a description for a position. Seat size can be adjusted, and seat types can be assigned for different pricing (VIP, 1st Class, 2nd Class, etc.).

Customization

Different seat plan designs and colors can be changed through settings. Prices can also be set differently based on ticket type. Additionally, extra products (like caps, t-shirts, etc.) can be sold alongside the seat booking.

Use Cases

This addon is suitable for any kind of seat plan booking that has a fixed time and date:

  • Seminar booking
  • Billboard booking
  • Trade fair position selling
  • Cinema booking (requires Recurring addon)
  • Movie ticket selling (requires Recurring addon)
  • Beach umbrella position selling
  • Restaurant table booking
