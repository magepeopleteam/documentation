Email Reminder Addon

Send automatic reminder emails before an event. This helps reduce no-shows and keeps attendees informed.

Installation

  1. Download the addon zip from your account.
  2. Go to Plugins → Add New → Upload Plugin.
  3. Upload the zip and click Install Now.
  4. Activate the plugin from the Plugins page.

Settings

You can set up a maximum of 3 reminders per event:

  • Each reminder can be configured with its own timing (before or after the event).
  • The addon keeps log details so you can view the sending history for each event.
