Membership Price — WpEvently Add‑On

A membership pricing addon for the event is needed for that organiser who wants to offer different pricing based on the user role of a WordPress website.

Overview

With this addon, the WooCommerce Event Manager Plugin's ticket price can be set as different member prices. Pricing works depending on the member type logged in.

Compatibility

Currently supported with:

  • WordPress Default User Roles
  • WooCommerce Membership Plugin

Settings

After installation, navigate to the event settings to select which membership plugin you are using. Then configure membership pricing for each ticket type — different prices can be set for different membership levels.
