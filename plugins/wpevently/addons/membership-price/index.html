Membership Price Addon

A membership pricing addon for the event is needed for that organiser who wants to offer different pricing based on the user role of a WordPress website.

Installation

  1. Download the addon zip from your account.
  2. Go to Plugins → Add New → Upload Plugin.
  3. Upload the zip and click Install Now.
  4. Activate the plugin from the Plugins page.

Compatibility

Currently supported with:

  • WordPress Default User Roles
  • WooCommerce Membership Plugin

Settings

  1. Navigate to the event settings and select which membership plugin you are using.
  2. Configure membership pricing for each ticket type — different prices can be set for different membership levels.
