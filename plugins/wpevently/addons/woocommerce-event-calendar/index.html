Overview

The WooCommerce Event Calendar Addon showcases events into an awesome calendar display. Just install the plugin and use the shortcode to show the calendar anywhere on your website.

Shortcode

Display the current month calendar:

[mep-event-calendar]

To display a specific month, use the shortcode with the year and month:

[mep-event-calendar-month month='2028-09']
