WooCommerce Event Waitlist — WpEvently Add‑On

Waitlist addon is very useful, if any ticket became sold out organiser can get subscription for next available ticket sell.

Overview

The WooCommerce Event Waitlist Addon works with the Event plugin. Once event ticket available quantity finishes, the waitlist plugin activates and subscribers can join the waitlist. The admin can then send email to users for the next free slot.

Installation

Install and activate the addon from your WordPress plugins page. Once activated, the waitlist feature will be available on events where ticket quantity has reached zero.

Usage

When an event ticket sells out, customers see a waitlist signup option instead of the add-to-cart button. Subscribers are added to the waitlist, and the admin receives notification. When a new slot becomes available, the admin can send email notifications to waitlisted users.
