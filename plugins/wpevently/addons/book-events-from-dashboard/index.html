Book Events from Dashboard Addon

This Addon is mainly used for admin, if anyone wants to get order offline like phone order then after taking order admin can add attendee from dashboard.

Installation

  1. Download the addon zip from your account.
  2. Go to Plugins → Add New → Upload Plugin.
  3. Upload the zip and click Install Now.
  4. Activate the plugin from the Plugins page.

Usage

  1. After installation, find the new Book Event option in your WordPress admin.
  2. Select the event you want to book.
  3. Choose ticket types and quantities.
  4. Complete the booking on behalf of the customer — all from the dashboard.
