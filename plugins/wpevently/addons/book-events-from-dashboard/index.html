Overview

This addon adds a new feature to your Event Manager plugin so you can book any event from the WordPress Dashboard for your customer. This is mainly needed for phone call orders, offline orders, or spot registrations.

Usage

After installation, you will find a new "Book Event" option in your WordPress admin. You can select the event, choose ticket types and quantities, and complete the booking on behalf of the customer — all from the dashboard.
