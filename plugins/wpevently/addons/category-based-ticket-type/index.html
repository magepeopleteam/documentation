Overview

The Category-Based Ticket Type Addon allows you to organize and manage ticket types based on event categories. This makes it easier to apply consistent ticket configurations across events that share the same category.

Usage

After installing and activating the addon, go to the event category settings. You will find additional options to define default ticket types and pricing for each category. When creating a new event under that category, the ticket types will be pre-populated automatically.
