Event Duplicator — WpEvently Add‑On

Duplicator addon mainly needed to save time while event creating, if you have an already created event and you want to duplicate that event then this addon needed.

Overview

This addon adds a Duplicator feature to the WooCommerce Event Plugin. You can easily duplicate any event and re-use it, saving time when creating similar events.

Usage

After installation, go to the Events list in your WordPress admin. Hover over any existing event and you will see a "Duplicate" option. Clicking it creates an exact copy of the event, including all settings, ticket types, and details, which you can then modify as needed.
