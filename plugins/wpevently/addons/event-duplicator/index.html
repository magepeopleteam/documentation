Event Duplicator Addon

Duplicator addon mainly needed to save time while event creating, if you have an already created event and you want to duplicate that event then this addon needed.

Installation

  1. Download the addon zip from your account.
  2. Go to Plugins → Add New → Upload Plugin.
  3. Upload the zip and click Install Now.
  4. Activate the plugin from the Plugins page.

Usage

  1. Go to the Events list in your WordPress admin.
  2. Hover over any existing event to reveal the Duplicate option.
  3. Click Duplicate to create an exact copy of the event, including all settings, ticket types, and details.
  4. Modify the duplicated event as needed.
