Installation Methods

You have three options to install WooCommerce Event Manager.

Automatic Install via WordPress (Recommended)

  1. Go to your admin page and go to Plugins → Add New.
  2. Search for “evently” or find “WooCommerce evently” by Mage People team and click Install.
  3. Activate it from the installation page or your Plugins page.

Adding via Zip file

  1. Download the plugin zip from the WordPress directory.
  2. Go to Plugins → Add New, click Upload Plugin, choose the zip, and click Install Now.
  3. Activate the plugin from the installation or Plugins page.

Uploading via FTP

  1. Download the plugin zip from the WordPress directory.
  2. Connect via FTP, extract the zip, and upload the mage-eventpress folder to wp-content/plugins/.
  3. Go to your Plugins page and activate the plugin.

Pro Version Installation

Event Manager Pro Version comes with 2 addons:

  1. WooCommerce Event Manager Addon: Form Builder
  2. WooCommerce Event Manager Addon: PDF Tickets

Install WooCommerce Event Manager Addon: Form Builder

Download the plugin zip file from My Account after buying pro version then go to Plugins → Add New. Click on the Upload Plugin button. Choose the plugin zip file woocommerce-event-manager-addon-form-builder.zip and click on Install Now button.

Install WooCommerce Event Manager Addon: PDF Tickets

It is a similar process to install the form builder addon. In this case just choose the plugin zip file woocommerce-event-manager-pdf-ticket.zip and click on Install Now button.
