Template Selection for Single Events

Learn how to select and apply different templates to individual event pages. You can assign a unique look to each event or use global templates for consistency.

Selecting Templates

  • Edit a single event post in the WordPress admin
  • Look for the Template dropdown in the Page Attributes metabox
  • Choose from Default Template, Full Width, or custom Elementor templates
  • Save the event and view the front end to verify the template
  • Different events can use different templates as needed

Global Template Assignment

  • Use Elementor Pro Theme Builder for site-wide event templates
  • Set display conditions to apply templates to all events
  • Create category-specific templates for different event types
  • Override global templates on individual events when needed
  • Test templates with sample events before publishing

Common Configurations

  • Grid with 3 columns showing 9 events: [mep_event_grid column="3" limit="9"]
  • Category-filtered list: [mep_event_list category="workshop"]
  • Calendar in week view: [mep_calendar view="week"]
  • Carousel with 4 visible slides and autoplay: [mep_event_carousel slide="4" autoplay="yes"]
  • Minimal list limited to 5 items: [mep_event_minimal limit="5"]
