## Overview

Add QR code support to your bus ticket booking system for fast and secure ticket validation.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Settings

To enable QR code, go to **Bus → Settings → QR Settings**:

- **Enable/Disable** — Toggle the QR code feature on or off.
- **Checker Role** — Set the user role for ticket checkers who can scan and validate tickets.

## Usage

1. Once enabled, QR codes appear on all PDF tickets.
2. The checker scans the ticket using a QR scanning app.
3. Check-in status is updated in real time and can be viewed in the Passenger List.
