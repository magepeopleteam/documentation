## Overview

Offer bulk discounts for group travelers. Attract bigger groups and increase sales, while travelers enjoy cost-effective packages.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Usage

1. Navigate to the tour edit page and open **Group & Bulk Pricing** settings.
2. Define pricing tiers based on the number of travelers.
3. Set discount percentages or fixed prices for each tier.
4. Bulk pricing is automatically applied at checkout based on the group size.
