## Overview

Control tour participation with minimum and maximum limits. Ensures every tour has the right number of travelers, helping you plan better and avoid overbooking.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Usage

1. Go to the tour edit page and find **Max-Min Booking** settings.
2. Set the minimum number of travelers required for the tour to run.
3. Set the maximum capacity for the tour.
4. Customers cannot book below the minimum or above the maximum limit.
