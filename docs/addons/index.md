# MagePeople Addons

Extend your MagePeople plugins with powerful addons that add new features and functionality.

Browse addons by plugin:

- **[eCab Addons](/docs/addons/ecab/)** — Distance tier pricing, peak hour pricing
- **[WPBusTicketly Addons](/docs/addons/wpbusticketly/)** — QR code integration
- **[WPEvently Addons](/docs/addons/wpevently/)** — Event seat plan, email reminders, QR codes, coupons, waitlist, and more
- **[WPRently Addons](/docs/addons/wprently/)** — Seasonal pricing, min-max booking, multi-day discounts, backend orders
- **[WPTravelly Addons](/docs/addons/wptravelly/)** — Seat plan, seasonal pricing, group pricing, QR codes, early bird pricing, and more
