# Bank Payment Gateway

Add direct bank transfer as a payment option for your MagePeople booking plugins.

## Features

- Direct bank transfer payment
- Multiple bank account support
- Custom payment instructions
- Automatic order status updates
- Compatible with all MagePeople plugins

## Installation

1. Download the addon from your MagePeople account.
2. Install and activate via **Plugins → Add New**.
3. Configure bank account details in **WooCommerce → Settings → Payment Gateways**.
