# Settings

## General Settings

- **Enable/Disable** — Turn the gateway on or off
- **Title** — Payment method title displayed at checkout
- **Description** — Payment method description

## Bank Account Details

- **Account Name** — Bank account holder name
- **Account Number** — Bank account number
- **Bank Name** — Name of the bank
- **IBAN** — International bank account number
- **SWIFT/BIC** — SWIFT code
- **Additional Instructions** — Extra payment instructions

## Order Settings

- **Order Status** — Default order status after payment
- **Pending Status** — Status for pending bank transfer orders
- **Instructions** — Payment instructions displayed on thank you page
