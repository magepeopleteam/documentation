# Widgets Reference

## Booking Form Widget

Display a complete booking form for any MagePeople plugin product.

### Settings

- **Select Product** — Choose the product to display
- **Form Style** — Default, compact, or full-width layout
- **Show Price** — Display price information
- **Show Availability** — Show real-time availability

## Search Form Widget

Display a search form for finding available products.

### Settings

- **Search Fields** — Enable/disable specific search fields
- **Button Text** — Customize search button label
- **Result Page** — Link to search results page

## Product Grid Widget

Display products in a customizable grid layout.

### Settings

- **Columns** — 2, 3, or 4 columns
- **Products Per Page** — Number of products to display
- **Category Filter** — Filter by product category
- **Show Pagination** — Enable/disable pagination

## Price Table Widget

Display pricing information in a table format.

### Settings

- **Layout** — Table or card layout
- **Highlight Feature** — Feature column to highlight
- **Show CTA** — Display call-to-action button
