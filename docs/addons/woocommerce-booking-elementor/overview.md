# WooCommerce Booking Elementor

Extend your MagePeople booking plugins with custom Elementor widgets for building beautiful booking pages.

## Features

- Drag-and-drop booking form widget
- Customizable search forms
- Product grid and list widgets
- Price table widgets
- Category display widgets
- Fully responsive designs

## Requirements

- Elementor (Free or Pro) installed
- Compatible MagePeople booking plugin installed

## Installation

1. Download the addon from your MagePeople account.
2. Install and activate via **Plugins → Add New**.
3. Open any page with Elementor to access the new widgets.
