## Overview

Set minimum-day and maximum-day booking limits to your rental system. Control how many days customers can rent an item, preventing too-short or too-long rentals.

## Installation

1. Visit the [Min and Max Booking Day Addon page](https://mage-people.com/product/min-and-max-booking-day-for-booking-and-rental-plugin/) and click **Purchase Now**.
2. Select your preferred plan and buy the addon.
3. You will receive a download link shortly.
4. Go to your WordPress dashboard and navigate to **Plugins → Add New → Upload Plugin**.
5. Upload the zip file and click **Install Now**.
6. After installation, click **Activate**.

## Usage

1. After activation, open any rental item from the All Rental Items list.
2. Locate the **Minimum and Maximum Day Rent Configuration** panel.
3. Set the **Minimum Booking Duration** (e.g., 1 day minimum).
4. Set the **Maximum Booking Duration** (e.g., 30 days maximum).
5. Optionally enable **date-wise Min/Max** to vary the limits by specific dates.
6. Click **Update** — the front-end will automatically enforce these limits.
7. When a customer tries to book outside these limits, the system displays an error message and prevents the booking.
