# Custom Checkout Fields

Add custom fields to the WooCommerce checkout page for collecting additional information during booking.

## Features

- Unlimited custom fields
- Multiple field types (text, select, checkbox, file upload)
- Conditional field logic
- Per-product field rules
- Compatible with all MagePeople plugins

## Installation

1. Download the addon from your MagePeople account.
2. Install and activate via **Plugins → Add New**.
3. Configure fields in **WooCommerce → Settings → Custom Checkout Fields**.
