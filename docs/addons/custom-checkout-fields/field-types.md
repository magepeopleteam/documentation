# Field Types

## Text Input

Standard single-line text field. Use for names, phone numbers, etc.

## Textarea

Multi-line text input for longer responses like special requests.

## Select Dropdown

Dropdown menu with predefined options. Use for choices like time slots, preferences.

## Checkbox

Single or multiple checkbox options. Use for agreements, add-ons.

## Radio Button

Single-selection radio button group.

## File Upload

Allow customers to upload files (IDs, documents, etc.).

## Date Picker

Calendar date selection field.

## Conditional Logic

Fields can be shown or hidden based on values of other fields, creating dynamic checkout forms.
