## Overview

Optimize pricing during peak hours to maximize revenue. Automatically adjust fares based on demand and time of day to ensure profitable operations.

## Installation

1. Download the addon zip from your MagePeople account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip file and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Usage

1. Go to the eCab settings page and locate the Peak Hour Pricing section.
2. Define peak hour time windows (e.g., 7:00–9:00 AM, 5:00–7:00 PM).
3. Set a multiplier or fixed surcharge for each window.
4. When a booking falls within a defined peak hour window, the surcharge is automatically applied to the fare at checkout.
