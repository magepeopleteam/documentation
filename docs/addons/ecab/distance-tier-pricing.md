## Overview

Charge accurately with tiered pricing based on distance. Create flexible fare rules to match short, medium, or long rides and increase your earning potential.

## Installation

1. Download the addon zip from your MagePeople account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip file and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Usage

1. Navigate to the eCab settings panel and locate the Distance Tier Pricing section.
2. Define distance ranges (e.g., 0–5 km, 5–15 km, 15–30 km).
3. Set a fixed or per-km price for each tier.
4. When a customer books a ride, the system automatically calculates the distance and applies the corresponding tier pricing during checkout.
