## Overview

Enable customers to purchase multiple tickets in groups with flexible quantity selection options.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Settings

After installation, navigate to the event edit page to enable common/global quantity management:

- Instead of tracking stock per ticket type, the addon uses a single pooled quantity for the entire event.
