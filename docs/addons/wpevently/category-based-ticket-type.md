## Overview

Assign different ticket types based on categories for better organization and flexible event pricing options.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Usage

1. Go to the **Event Category** settings.
2. Configure default ticket types and pricing for each category.
3. Create a new event under that category — ticket types will be pre-populated automatically.
