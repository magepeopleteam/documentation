## Overview

Waitlist addon is very useful, if any ticket became sold out organiser can get subscription for next available ticket sell.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Usage

1. When an event ticket sells out, customers see a **waitlist signup** option instead of the add-to-cart button.
2. Subscribers are added to the waitlist and the admin receives a notification.
3. When a new slot becomes available, the admin can send email notifications to waitlisted users.
