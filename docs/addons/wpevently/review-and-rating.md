## Overview

The Event Review and Rating Addon is a fantastic value-added offering designed to help businesses gather feedback from customers about their events.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Settings

After installation, go to the event edit page:
- Enable the review and rating section for that specific event.
- Control which user roles are allowed to submit reviews and ratings.

## Review Dashboard

The admin panel includes a review dashboard where you can manage all submitted reviews, approve or reject them, and monitor ratings across events.
