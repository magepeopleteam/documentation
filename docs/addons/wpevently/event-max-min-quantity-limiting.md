## Overview

The event max-min addon is necessary for limiting the amount of tickets sold. Organizers can set a maximum buying quantity or minimum purchase quantity.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Settings

After installation, navigate to the event edit page to configure quantity limits:

- Set separate minimum and maximum values for each ticket type.

## Frontend Display

When a customer tries to add tickets outside the allowed range, the addon displays a notification message indicating the quantity limits.
