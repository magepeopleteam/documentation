# Troubleshooting

## Common Issues

### Plugin Does Not Activate

- Ensure your WordPress version meets the minimum requirement.
- Check PHP version compatibility (7.4+ required).
- Verify no conflicting plugins are installed.

### Booking Form Not Showing

- Ensure the correct shortcode is used.
- Check that the product/service is published and has available slots.
- Verify Elementor widget is properly configured (if using Elementor).

### Emails Not Sending

- Configure an SMTP plugin for reliable delivery.
- Check email settings in the plugin configuration.
- Verify the sender email address is valid.

### Payment Gateway Issues

- Ensure your payment gateway is configured correctly in WooCommerce.
- Verify SSL certificate is installed on your site.
- Check gateway logs in WooCommerce → Status → Logs.
