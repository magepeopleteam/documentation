# Installation

Follow these steps to install any MagePeople plugin on your WordPress site.

## Prerequisites

- WordPress 5.8 or higher
- PHP 7.4 or higher
- WooCommerce 7.0 or higher (for WooCommerce-based plugins)

## Via WordPress Admin

1. Log in to your WordPress admin dashboard.
2. Navigate to **Plugins → Add New**.
3. Click **Upload Plugin**.
4. Choose the plugin ZIP file and click **Install Now**.
5. Click **Activate** after installation completes.

<video src="/video/plugins/wpevently/upload.webm" controls width="100%" class="doc-video"></video>

## Via FTP

1. Extract the plugin ZIP file on your computer.
2. Upload the extracted folder to `/wp-content/plugins/`.
3. Log in to WordPress admin and go to **Plugins → Installed Plugins**.
4. Find the plugin and click **Activate**.

<video src="/video/plugins/wpevently/ftp.webm" controls width="100%" class="doc-video"></video>

## Pro Version Installation

Download the plugin zip from your [MagePeople account](https://mage-people.com/my-account/) after purchasing, then:

1. Go to **Plugins → Add New**, click **Upload Plugin**, choose the zip, and click **Install Now**.
2. Activate the plugin from the installation or **Plugins** page.
3. Install and activate any required addon plugins (e.g., PDF Support for ticket generation).

## Post-Installation

After activation, you'll find the plugin settings in your WordPress admin menu. Refer to the individual plugin documentation for configuration details.
