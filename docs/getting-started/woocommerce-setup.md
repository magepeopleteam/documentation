# WooCommerce Setup

Many MagePeople plugins extend WooCommerce. Follow this guide to configure WooCommerce properly.

## Install WooCommerce

1. Go to **Plugins → Add New**.
2. Search for "WooCommerce".
3. Install and activate the official WooCommerce plugin.
4. Complete the WooCommerce setup wizard.

## Recommended Settings

- Enable **Geolocation** for accurate tax and shipping calculations.
- Set your **Default Customer Location** appropriately.
- Configure **Payment Gateways** that your customers will use.

## Required Pages

WooCommerce will automatically create the following pages:
- Shop
- Cart
- Checkout
- My Account

Ensure these pages are published and accessible.
