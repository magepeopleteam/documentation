# Elementor Integration

MagePeople plugins offer native Elementor widgets for building custom booking pages.

## Requirements

- Elementor (Free or Pro) installed and activated
- MagePeople plugin with Elementor support activated

## Available Widgets

Depending on the plugin, you'll have access to widgets such as:

- Booking Form
- Search Form
- Product Grid
- Price Table
- Category List

## How to Use

1. Edit a page or post with Elementor.
2. Search for the MagePeople widget in the Elementor panel.
3. Drag and drop the widget onto your page.
4. Configure the widget settings from the Elementor sidebar.
