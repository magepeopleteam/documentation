# Shortcodes

## Booking Form

```
[car_rental_booking_form]
```

Display vehicle booking form.

## Vehicle List

```
[car_rental_vehicles limit="6" columns="3"]
```

Display available vehicles in a grid.

## Search Form

```
[car_rental_search]
```

Display vehicle search widget.

## Availability Calendar

```
[car_rental_calendar vehicle_id="123"]
```

Display availability for a specific vehicle.
