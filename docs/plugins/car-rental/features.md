# Features

## Fleet Management

- Add vehicles with details, images, and specs
- Categorize by type (sedan, SUV, luxury, etc.)
- Track vehicle availability
- Maintenance scheduling

## Driver Management

- Driver profiles and assignment
- Driver availability tracking
- Commission management

## Booking System

- Online vehicle reservation
- Pickup and return location selection
- Real-time availability checking
- Deposit and full payment options
