# Settings

## General Settings

- **Vehicle Slug** — Custom URL slug
- **Default Currency** — Set pricing currency
- **Distance Unit** — Kilometers or miles

## Pricing Settings

- **Daily Rate** — Default daily rental rate
- **Weekly Rate** — Discounted weekly rate
- **Monthly Rate** — Discounted monthly rate
- **Security Deposit** — Required deposit amount

## Availability Settings

- **Buffer Time** — Time between rentals
- **Max Rental Days** — Maximum rental duration
- **Advance Booking** — Minimum notice required
