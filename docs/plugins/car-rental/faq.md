# FAQ

## Can I manage multiple vehicle types?

Yes. Add unlimited vehicles with different types and pricing.

## How does pricing work?

Daily, weekly, and monthly rates can be set per vehicle. Seasonal pricing is also supported.

## Can customers choose pickup locations?

Yes. Multiple pickup and return locations can be configured.
