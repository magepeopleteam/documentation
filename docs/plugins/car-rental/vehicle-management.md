# Vehicle Management

The Vehicle Management system allows you to build and maintain your entire fleet from the WordPress admin. Each vehicle profile supports images, detailed specifications, feature flags, categories, and custom terms.

## Adding and Managing Vehicles

All vehicles are managed under the Car Rental > Vehicles menu in the WordPress admin panel.

### Creating a Vehicle

- Go to Car Rental > Vehicles > Add New.
- Enter the vehicle name (e.g., "Toyota Corolla 2025").
- Upload a featured image and optional gallery images.
- Write a detailed description highlighting features and conditions.
- Select or create a vehicle category (e.g., Sedan, SUV, Hatchback).
- Assign vehicle features such as AC, GPS, Bluetooth, and transmission type.
- Set passenger capacity, number of doors, and luggage capacity.
- Publish the vehicle to make it visible on the front end.

### Editing & Organising

- Edit any vehicle from the Vehicles list table by clicking its title.
- Use quick edit for bulk changes to status, category, or featured status.
- Drag-and-drop reorder is available with the built-in sortable list.
- Filter the vehicle list by category, status (published/draft), or date.
- Trash vehicles to remove them from public view without permanent deletion.

## Vehicle Categories & Features

### Category Management

- Create categories under Car Rental > Vehicle Categories.
- Assign a name, slug, description, and optional image per category.
- Categories help users filter vehicles on the front-end listing.
- Nested categories are supported for sub-types (e.g., Luxury > Sports Car).
- Categories can be shown or hidden from the booking form filter.

### Feature Flags

- **Transmission:** Automatic or Manual.
- **Fuel Type:** Petrol, Diesel, Electric, Hybrid.
- **Comfort:** Air Conditioning, Heated Seats, Sunroof.
- **Technology:** Bluetooth, Apple CarPlay, Android Auto, GPS.
- **Safety:** ABS, Airbags, Parking Sensors, Rear Camera.
- Features display as icons or tags on the vehicle detail page.

## Extra Services & Add-ons

Attach optional extra services to any vehicle to upsell during the booking process.

### Managing Extras

- Go to Car Rental > Extras to create add-on services.
- Set a name (e.g., "Child Seat", "Additional Driver"), price, and description.
- Choose between one-time fee or per-day pricing.
- Extras can be optional or required during booking.
- Assign extras globally or per vehicle from the vehicle edit screen.

### Common Extra Services

- Insurance packages (collision, theft, liability).
- GPS navigation system rental.
- Child booster seat or baby seat.
- Additional driver authorization.
- Roadside assistance coverage.
- Winter tire / snow chain packages.

## Location Management

Configure pickup and drop-off locations for your fleet. Multi-location support allows customers to pick up from one location and return to another.

### Setting Up Locations

- Go to Car Rental > Locations > Add New.
- Enter the location name, address, and contact details.
- Set latitude and longitude for map integration.
- Define opening hours for each location.
- Mark a location as "Head Office" for primary business address.

### Assigning Locations to Vehicles

- Each vehicle can be available at one or multiple locations.
- Set default pickup and drop-off locations per vehicle.
- Customers can change locations during booking if enabled.
- Location availability affects vehicle search results.

## Import & Export

- Use the built-in CSV import tool to bulk-add vehicles from a spreadsheet.
- Export your fleet to CSV for backup or offline editing.
- The import supports vehicle name, description, category, features, pricing, and images.
- Export data includes all vehicle metadata and assigned extras.
