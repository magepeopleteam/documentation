# Shortcodes

## Group Booking Form

```
[wpgrouply_booking_form]
```

Display the group booking form.

## Participant Form

```
[wpgrouply_participants]
```

Display participant details collection form.

## Group Pricing Table

```
[wpgrouply_pricing]
```

Display tiered group pricing table.

## Group Dashboard

```
[wpgrouply_dashboard]
```

Display group booking dashboard for the customer.
