# WPGrouply

WPGrouply enables group bookings for MagePeople plugins, allowing multiple participants to book under a single reservation.

## Key Features

- Group booking support
- Per-person pricing
- Participant management
- Group discount rules
- WooCommerce integration
- Compatible with all MagePeople plugins

## Getting Started

Navigate to **WPGrouply** in your WordPress admin to configure group booking settings.
