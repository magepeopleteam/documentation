# Settings

## General Settings

- **Equipment Slug** — Custom URL slug
- **Default Rental Period** — Default rental duration
- **Time Format** — Pickup/return time format

## Rental Settings

- **Min Rental Period** — Minimum rental duration
- **Max Rental Period** — Maximum rental duration
- **Buffer Time** — Time between rentals for inspection
- **Late Return Fee** — Fee structure for late returns

## Deposit Settings

- **Require Deposit** — Enable deposit requirement
- **Deposit Type** — Percentage or fixed amount
- **Deposit Amount** — Default deposit value

## Notification Settings

- **Booking Confirmation** — Customer confirmation template
- **Return Reminder** — Reminder before rental end
- **Overdue Alert** — Notification for late returns
