# FAQ

## What types of equipment can I rent?

Any physical item — tools, machinery, sports equipment, camping gear, party supplies, etc.

## How is availability tracked?

Real-time availability is tracked per item based on bookings and manual blocks for maintenance.

## Can I set different prices for weekends?

Yes. Weekend, holiday, and seasonal pricing are supported.

## Is deposit collection supported?

Yes. Deposits can be configured as a percentage or fixed amount.

## Does it work with Elementor?

Yes. WPEquipments includes Elementor widgets.
