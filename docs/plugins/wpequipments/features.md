# Features

## Inventory Management

- Add equipment items with details
- Set stock quantity per item
- Categorize equipment by type
- Upload equipment images and manuals

## Rental Pricing

- Per-day, per-hour, and per-week rates
- Weekend and holiday pricing
- Security deposit configuration
- Late return fees

## Availability

- Real-time availability calendar
- Block unavailable dates
- Buffer time between rentals
- Maintenance scheduling

## Booking System

- Online equipment reservation
- Pickup and return time selection
- Deposit and full payment options
- Automated booking confirmations
