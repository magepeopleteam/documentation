# Shortcodes

## Equipment Grid

```
[wpequipments_grid limit="6" columns="3"]
```

Display equipment items in a grid.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| limit | int | 10 | Number of items |
| columns | int | 3 | Grid columns |
| category | string | all | Filter by category |

## Rental Form

```
[wpequipments_rental_form product_id="123"]
```

Display rental booking form for an equipment item.

## Availability Calendar

```
[wpequipments_calendar product_id="123"]
```

Display availability for a specific equipment item.

## Search Form

```
[wpequipments_search]
```

Display equipment search form.
