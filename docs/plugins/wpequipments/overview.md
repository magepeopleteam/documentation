# WPEquipments

WPEquipments is a specialized equipment rental plugin for WordPress, designed for businesses renting tools, machinery, sports equipment, and more.

## Key Features

- Equipment inventory management
- Daily/hourly/weekly rental pricing
- Availability calendar
- Maintenance tracking
- WooCommerce integration
- Elementor widgets

## Getting Started

Go to **WPEquipments** in your WordPress admin to start managing equipment inventory.
