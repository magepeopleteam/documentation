# Features

## Event Management

- Create single or recurring events
- Set event dates, times, and locations
- Add speakers and organizers
- Custom event categories and tags

## Ticket System

- Multiple ticket types (General, VIP, Early Bird)
- Tiered pricing and discounts
- Ticket quantity limits
- Printable ticket PDFs

## Display Options

- Calendar view with month/week/day navigation
- Grid and list layouts
- Featured events carousel
- Elementor widgets for custom layouts

## Payment & Checkout

- WooCommerce integration for secure payments
- Support for all WooCommerce payment gateways
- Tax and fee configuration
- Discount coupon support
