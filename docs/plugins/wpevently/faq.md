# FAQ

## Can I sell tickets for free events?

Yes. Set the ticket price to $0 for free events.

## Does WPEvently support recurring events?

Yes. You can create daily, weekly, monthly, or custom recurring events.

## Can I use WPEvently with any theme?

WPEvently works with most WordPress themes. For best results, use a theme that supports WooCommerce.

## Is Elementor supported?

Yes. WPEvently includes dedicated Elementor widgets for building custom event pages.

## How do I issue refunds?

Process refunds through WooCommerce order management. The ticket status updates automatically.
