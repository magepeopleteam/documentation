# Settings

## General Settings

- **Event Slug** — Customize the URL slug for events
- **Event Per Page** — Number of events to display per page
- **Default Event Image** — Set a fallback image for events

## Payment Settings

- **Currency** — Choose the currency for ticket prices
- **Tax Settings** — Configure tax rates
- **Payment Gateways** — Enable/disable WooCommerce gateways

## Email Settings

- **Admin Notification** — Receive email on new bookings
- **Customer Confirmation** — Send confirmation to customers
- **Reminder Emails** — Schedule reminder emails before events

## Display Settings

- **Default Layout** — Choose between calendar, grid, or list
- **Calendar Start Day** — Set the first day of the week
- **Time Format** — 12-hour or 24-hour format
