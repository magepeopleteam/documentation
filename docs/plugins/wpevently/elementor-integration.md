# Elementor Integration

## Using Event Widgets in Elementor

First, edit any page with Elementor and scroll down the Elementor widgets panel. You'll find the WooCommerce Event Manager widgets at the bottom of the panel.

### Step-by-Step Guide

1. Edit any page or template with Elementor.
2. In the Elementor panel, scroll down to the **WooCommerce Event Manager** section.
3. Drag and drop any event widget onto your page canvas.
4. Configure the widget settings in the left panel — adjust layout, filters, and styling options.
5. Click **Publish** or **Update** to save your changes.

## How to Add the Shortcode in Elementor

You can use the native Elementor Shortcode widget to embed any WooCommerce Event Manager shortcode directly into your page or template.

1. Edit any page or template with Elementor.
2. Search for the **Shortcode** widget in the Elementor panel.
3. Drag it onto the canvas.
4. Paste your shortcode into the widget input field (e.g. `[event-list style='grid']`).
5. Click Apply — the events render directly in the editor.
