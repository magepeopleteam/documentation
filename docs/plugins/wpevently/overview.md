# WPEvently

WPEvently is a powerful WordPress event management plugin that lets you create, manage, and sell tickets for events of any size.

## Key Features

- Create unlimited events
- Sell tickets online via WooCommerce
- Event calendar and grid layouts
- Recurring events support
- Elementor integration
- Multiple ticket types and pricing tiers

## Getting Started

After installation and activation, navigate to **Events** in your WordPress admin to start creating events.
