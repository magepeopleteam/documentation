# Installation & Setup

## Installation Methods

You have three options to install WooCommerce Event Manager.

### Automatic Install via WordPress (Recommended)

1. Go to your admin page and go to **Plugins → Add New**.
2. Search for "evently" or find "WooCommerce evently" by **Mage People** team and click **Install**.
3. Activate it from the installation page or your **Plugins** page.

<video src="/video/plugins/wpevently/install.webm" controls width="100%" class="doc-video"></video>

### Adding via Zip file

1. Download the plugin zip from the WordPress [directory](https://wordpress.org/plugins/mage-eventpress/).
2. Go to **Plugins → Add New**, click **Upload Plugin**, choose the zip, and click **Install Now**.
3. Activate the plugin from the installation or **Plugins** page.

<video src="/video/plugins/wpevently/upload.webm" controls width="100%" class="doc-video"></video>

### Uploading via FTP

1. Download the plugin zip from the WordPress [directory](https://wordpress.org/plugins/mage-eventpress/).
2. Connect via FTP, extract the zip, and upload the `mage-eventpress` folder to `wp-content/plugins/`.
3. Go to your **Plugins** page and activate the plugin.

<video src="/video/plugins/wpevently/ftp.webm" controls width="100%" class="doc-video"></video>

## Pro Version Installation

Download the plugin zip from your account after purchasing, then:

1. Go to **Plugins → Add New**, click **Upload Plugin**, choose the zip, and click **Install Now**.
2. Activate the plugin from the installation or **Plugins** page.
3. Install and activate the **PDF Support** plugin for generating PDF tickets.
