# WPFoodly

WPFoodly is a food ordering plugin for WordPress, enabling restaurants, cafes, and food businesses to accept online orders.

## Key Features

- Menu management
- Restaurant profiles
- Online food ordering
- Delivery and pickup options
- WooCommerce payment integration
- Elementor widgets

## Getting Started

Navigate to **WPFoodly** in your WordPress admin to set up your restaurant and menu.
