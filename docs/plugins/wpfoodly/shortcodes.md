# Shortcodes

## Menu Display

```
[wpfoodly_menu location_id="1"]
```

Display the restaurant menu.

## Order Form

```
[wpfoodly_order_form]
```

Display the food ordering form.

## Restaurant Info

```
[wpfoodly_info location_id="1"]
```

Display restaurant information and hours.

## Delivery Check

```
[wpfoodly_delivery_check]
```

Check if delivery is available for an address.

## Cart

```
[wpfoodly_cart]
```

Display the order cart.
