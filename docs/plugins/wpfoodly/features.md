# Features

## Menu Management

- Create food categories and items
- Set item prices and variations
- Add modifiers (extras, size options)
- Upload food images
- Daily specials and featured items

## Restaurant Setup

- Multiple restaurant/location support
- Operating hours per location
- Delivery zones and fees
- Minimum order amounts

## Ordering System

- Online menu browsing
- Cart and checkout flow
- Delivery time slot selection
- Order notes and special instructions

## Order Management

- New order notifications
- Order status tracking
- Print receipts and invoices
- Order history and analytics
