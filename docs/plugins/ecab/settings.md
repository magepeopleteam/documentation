# Settings

## General Settings

- **Cab Slug** — Custom URL slug for cab listings
- **Distance Unit** — Kilometers or miles
- **Default Currency** — Set your operating currency
- **Booking Lead Time** — Minimum advance booking time

## Pricing Settings

- **Base Fare** — Starting fare for all trips
- **Per Kilometer Rate** — Distance-based charge
- **Per Minute Rate** — Time-based waiting charge
- **Cancellation Fee** — Charge for cancelled bookings

## Map Settings

- **Google Maps API Key** — Required for location selection
- **Default Map Center** — Set default map coordinates
- **Auto Complete Address** — Enable address autocomplete

## Notification Settings

- **Driver Notifications** — Alert drivers of new bookings
- **Customer Notifications** — Booking confirmation and updates
- **Admin Notifications** — New booking alerts
