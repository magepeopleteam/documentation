# FAQ

## How does pricing work?

Pricing is based on distance, calculated from the base fare plus per-kilometer charges. Additional charges apply for waiting time and peak hours.

## Can drivers manage their own profiles?

Yes. Drivers can be given front-end access to manage their availability and view assigned bookings.

## Does eCab support multiple currencies?

Yes. eCab supports any currency configured in WooCommerce.

## Can I set different prices for different cab types?

Yes. Each cab type can have its own pricing structure.

## Is Google Maps required?

Google Maps is used for location selection and fare calculation. An API key is required.
