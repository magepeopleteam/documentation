# Features

## Fleet Management

- Add and manage cab types (sedan, SUV, luxury, etc.)
- Set cab capacity, luggage limits, and features
- Upload cab images and details
- Track cab availability in real-time

## Driver Management

- Add driver profiles with photos
- Assign drivers to specific cabs
- Track driver availability
- Driver commission management

## Booking System

- Real-time availability checking
- Distance-based fare calculation
- Waiting time charges
- Multiple stop support

## Pricing

- Per-kilometer/mile pricing
- Base fare + distance model
- Peak hour surcharges
- Night trip rates
