# Shortcodes

## Booking Form

```
[ecab_booking_form]
```

Display the cab booking form.

## Cab Search

```
[ecab_search]
```

Display a cab search widget.

## Cab List

```
[ecab_list limit="10" columns="3"]
```

Display available cabs in a grid.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| limit | int | 10 | Number of cabs to show |
| columns | int | 3 | Grid columns (2, 3, or 4) |

## Fare Calculator

```
[ecab_fare_calculator]
```

Display an interactive fare calculator.
