# eCab

eCab is a comprehensive taxi booking plugin for WordPress that allows you to manage cabs, drivers, and ride bookings seamlessly.

## Key Features

- Cab fleet management
- Driver assignment and tracking
- Online ride booking via WooCommerce
- Distance-based pricing
- Multiple pickup/dropoff locations
- Elementor integration

## Getting Started

Navigate to **eCab** in your WordPress admin to start managing your fleet.
