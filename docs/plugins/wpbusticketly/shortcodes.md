# Shortcodes

## Ticket Booking Form

```
[wpbusticketly_booking_form]
```

Display the bus ticket booking form.

## Route Search

```
[wpbusticketly_search]
```

Display the route search widget.

## Seat Map

```
[wpbusticketly_seat_map schedule_id="123"]
```

Display the interactive seat selection map.

## Bus Schedule

```
[wpbusticketly_schedule route_id="123"]
```

Display schedule for a specific route.

## Ticket Price

```
[wpbusticketly_price from="city1" to="city2"]
```

Display ticket price between two stops.
