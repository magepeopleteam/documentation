# Features

## Bus Management

- Add bus types and capacities
- Seat layout configuration
- Bus amenities and features
- Fleet tracking

## Route Management

- Create routes with multiple stops
- Distance and duration calculation
- Route pricing per stop
- Multiple schedules per route

## Seat Selection

- Interactive seat map
- Reserved and available seats
- Gender-based seating option
- Wheelchair-accessible seats

## Booking System

- Real-time seat availability
- E-ticket generation
- Boarding point selection
- Cancellation and refund management
