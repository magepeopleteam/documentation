# Settings

## General Settings

- **Ticket Slug** — Custom URL slug
- **Default Currency** — Set pricing currency
- **Time Format** — 12-hour or 24-hour
- **Date Format** — Display date format

## Booking Settings

- **Advance Booking Days** — How far ahead tickets can be booked
- **Booking Cut-off** — Hours before departure
- **Cancellation Policy** — Configure refund rules
- **Max Tickets Per Booking** — Limit per transaction

## Seat Settings

- **Default Seat Layout** — Configure standard seat map
- **Seat Color Coding** — Available, booked, blocked colors
- **Gender Seating** — Enable/disable gender-based seating

## Payment Settings

- **Payment Gateways** — Available payment methods
- **Tax Configuration** — Apply taxes to ticket prices
- **Service Fee** — Additional booking fee
