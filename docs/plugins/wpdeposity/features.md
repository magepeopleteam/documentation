# Features

## Deposit Types

- Percentage-based deposits (e.g., 30% upfront)
- Fixed amount deposits
- Full payment option
- Per-product deposit rules

## Payment Schedule

- Set payment due dates
- Multiple installment support
- Automatic payment reminders
- Late payment handling

## Balance Payments

- Automatic balance calculation
- Partial payment tracking
- Payment history per booking
- Manual balance marking

## Compatibility

- Works with WPEvently, eCab, WPTravelly, and all MagePeople plugins
- WooCommerce payment gateway support
- Compatible with WooCommerce Subscriptions
