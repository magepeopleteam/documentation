# WPDeposity

WPDeposity enables deposit and partial payment functionality for WooCommerce-based booking plugins, allowing customers to secure bookings with a deposit.

## Key Features

- Partial and full deposit options
- Percentage or fixed amount deposits
- Payment deadline management
- Multi-instance payment tracking
- WooCommerce integration
- Compatible with all MagePeople plugins

## Getting Started

Navigate to **WPDeposity** in your WordPress admin to configure deposit rules.
