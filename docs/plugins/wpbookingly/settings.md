# Settings

## General Settings

- **Booking Slug** — Custom URL slug
- **Time Slot Interval** — Duration of each time slot
- **Minimum Notice** — Advance booking required time
- **Maximum Advance** — How far ahead bookings are accepted

## Working Hours

- **Business Days** — Select operating days
- **Opening Hours** — Start and end time per day
- **Lunch Break** — Configure break times
- **Holidays** — Add holiday closures

## Payment Settings

- **Deposit Amount** — Require deposit at booking
- **Payment Gateways** — Available payment methods
- **Cancellation Fee** — Charge for late cancellations

## Notification Settings

- **Reminder Timing** — When to send reminders
- **Email Templates** — Customize notification emails
- **Admin Alerts** — New booking notifications
