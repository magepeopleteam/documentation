# Shortcodes

## Booking Form

```
[wpbookingly_booking_form]
```

Display the appointment booking form.

## Service List

```
[wpbookingly_services columns="3"]
```

Display available services in a grid.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| columns | int | 3 | Grid columns |

## Staff Directory

```
[wpbookingly_staff]
```

Display staff members and their availability.

## Availability Calendar

```
[wpbookingly_calendar staff_id="1"]
```

Show availability for a specific staff member.

## Customer Dashboard

```
[wpbookingly_dashboard]
```

Display customer booking history and management.
