# Features

## Service Management

- Create unlimited services
- Set duration and pricing per service
- Add service descriptions and images
- Categorize services

## Staff Management

- Add staff members with profiles
- Set working hours per staff
- Assign services to specific staff
- Staff availability calendar

## Booking Calendar

- Real-time slot availability
- Multiple calendar views
- Buffer time between appointments
- Break time management

## Customer Management

- Customer profiles and history
- Automated email/SMS reminders
- Self-service cancellation/rescheduling
- Booking notes for each appointment
