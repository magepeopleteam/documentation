# Settings

## General Settings

- **Rental Slug** — Custom URL slug
- **Default Rental Period** — Default duration for rentals
- **Time Format** — Choose pickup/return time format

## Pricing Settings

- **Default Rate Type** — Daily, hourly, or both
- **Tax Settings** — Apply tax to rental prices
- **Security Deposit** — Default deposit percentage

## Availability Settings

- **Buffer Time** — Time between rentals for cleaning/maintenance
- **Max Rental Days** — Maximum rental duration
- **Advance Booking** — Minimum notice for booking

## Notification Settings

- **Booking Confirmation** — Customer confirmation template
- **Return Reminder** — Reminder before rental end
- **Overdue Notification** — Alert for late returns
