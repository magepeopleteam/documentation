# Features

## Rental Products

- Convert any WooCommerce product to rental
- Set daily, hourly, or weekly rates
- Configure availability calendar
- Set minimum and maximum rental periods

## Pricing Options

- Per-day and per-hour pricing
- Weekend/weekday rate differentiation
- Seasonal pricing
- Security deposit amounts

## Inventory Management

- Real-time availability tracking
- Quantity-based inventory
- Pickup and return date management
- Maintenance scheduling

## Booking System

- Online rental booking via WooCommerce
- Deposit and full payment options
- Extend rental period from dashboard
- Late return fee calculation
