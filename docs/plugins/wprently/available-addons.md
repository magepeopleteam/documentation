# Available Addons

Extend the WpRently plugin with premium addons that add specialized features for booking limits, seasonal pricing, multi-day discounts, backend order management, and tiered discounts.

## Addon Overview

Each addon is a separate installable plugin that integrates seamlessly into the WpRently ecosystem. Addons are available for purchase individually from MagePeople.

### Min and Max Booking Limit

- [Min and Max Booking Limit Documentation](/addons/wprently/min-max-booking-limit)
- Set minimum-day and maximum-day booking limits for your rental system
- Control how many days customers can rent an item, preventing too-short or too-long rentals
- Configured per rental item from the item edit screen

### Seasonal Pricing Management

- [Seasonal Pricing Management Documentation](/addons/wprently/seasonal-pricing-management)
- Apply different rental prices based on seasons, dates, or time ranges
- Helps adjust pricing dynamically during peak and off-peak periods
- Rules can be applied globally or per rental item

### Multi-Day Discount Pricing

- [Multi-Day Discount Pricing Documentation](/addons/wprently/multi-day-discount-pricing)
- Offer special discounts for multi-day rentals
- Encourages longer bookings and increases overall rental value
- Configure discount tiers based on the number of rental days

### Pricing Discount Over x Days

- [Pricing Discount Over x Days Documentation](/addons/wprently/pricing-discount-over-x-days)
- Automatically apply tiered discounts as the rental duration increases
- Set discount percentages that scale with longer booking periods
- Compatible with all pricing models

### Admin Backend Order

- [Admin Backend Order Documentation](/addons/wprently/admin-backend-order)
- Create and manage rental orders directly from the WordPress dashboard
- Place bookings manually for customers who order by phone or in person
- Adds a **Purchase Order** menu under the Rent Item admin menu

## Addon Installation

### Installing an Addon

- Download the addon zip file from your MagePeople account after purchase
- Go to **Plugins → Add New → Upload Plugin**
- Choose the addon zip file and click **Install Now**
- Activate the addon after installation completes
- The addon's settings appear on the rental item edit screen or its own admin menu, depending on the addon

### License Management

- Each addon requires a valid license key for updates and support
- Enter the license key under WpRently → License for the addon
- Expired licenses stop receiving updates but continue to function

## Addon Dependencies

- All addons require the free WpRently plugin as a base
- Addons are compatible with each other and can be used simultaneously
