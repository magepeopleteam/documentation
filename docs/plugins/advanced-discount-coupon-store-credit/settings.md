# Settings

## General Settings

- **Coupon Prefix** — Prefix for auto-generated codes
- **Default Coupon Type** — Default discount type
- **Usage Limit** — Default usage limits

## Rule Settings

- **Combine Rules** — Allow stacking multiple rules
- **Priority Order** — Rule application priority
- **Exclude Sale Items** — Prevent discounts on sale items

## Display Settings

- **Show Discount on Cart** — Display savings in cart
- **Coupon Popup** — Enable coupon offer popup
- **Checkout Notice** — Show discount notice at checkout

## Notification Settings

- **Expiry Reminder** — Notify customers before coupon expiry
- **Admin Alerts** — Alert when coupon is used
