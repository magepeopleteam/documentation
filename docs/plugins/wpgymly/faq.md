# FAQ

## Can I create different membership plans?

Yes. Create unlimited membership plans with different durations, pricing, and features.

## Does WPGymly support class scheduling?

Yes. Schedule recurring classes with capacity limits and waitlist functionality.

## Can members book classes online?

Yes. Members can book classes through the front-end class schedule.

## Are trainer profiles supported?

Yes. Add trainer profiles with photos, bios, and class assignments.

## Does it work with Elementor?

Yes. WPGymly includes Elementor widgets for class schedules and membership plans.
