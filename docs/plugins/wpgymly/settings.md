# Settings

## General Settings

- **Gym Slug** — Custom URL slug
- **Time Format** — 12-hour or 24-hour
- **Currency** — Set pricing currency
- **Week Start Day** — First day of the week

## Membership Settings

- **Trial Period** — Free trial duration
- **Auto-renewal** — Enable/disable auto-renewal
- **Cancellation Policy** — Notice period for cancellation
- **Freeze Period** — Membership pause option

## Class Settings

- **Class Duration** — Default class length
- **Early Booking** — How far ahead classes can be booked
- **Cancellation Window** — Timeframe for class cancellation
- **Waitlist Size** — Maximum waitlist per class

## Payment Settings

- **Payment Gateways** — Available payment methods
- **Tax Settings** — Apply tax to memberships
- **Refund Policy** — Configure refund rules
