# Shortcodes

## Class Schedule

```
[wpgymly_schedule]
```

Display the class schedule.

## Membership Plans

```
[wpgymly_plans columns="3"]
```

Display membership plans in a grid.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| columns | int | 3 | Grid columns |

## Trainer Directory

```
[wpgymly_trainers]
```

Display trainer profiles.

## Booking Form

```
[wpgymly_booking_form class_id="123"]
```

Display booking form for a specific class.

## Member Dashboard

```
[wpgymly_dashboard]
```

Display member dashboard with bookings and membership info.
