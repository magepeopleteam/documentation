# WPGymly

WPGymly is a comprehensive gym and fitness studio management plugin for WordPress.

## Key Features

- Membership management
- Class scheduling
- Trainer profiles
- Package and plan creation
- WooCommerce payment integration
- Elementor widgets

## Getting Started

Navigate to **WPGymly** in your WordPress admin to configure your gym.
