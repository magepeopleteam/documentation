# Shortcodes

## Tour Grid

```
[wptravelly_tours limit="6" columns="3"]
```

Display tours in a grid layout.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| limit | int | 10 | Number of tours |
| columns | int | 3 | Grid columns |
| category | string | all | Filter by category |

## Tour Search

```
[wptravelly_search]
```

Display a tour search form.

## Featured Tours

```
[wptravelly_featured limit="4"]
```

Display featured tours.

## Tour Calendar

```
[wptravelly_calendar]
```

Display availability calendar for tours.

## Booking Form

```
[wptravelly_booking_form tour_id="123"]
```

Display booking form for a specific tour.
