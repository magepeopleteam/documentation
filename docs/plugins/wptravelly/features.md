# Features

## Tour Management

- Create unlimited tours and packages
- Multi-day and fixed-date tours
- Recurring tour schedules
- Custom tour categories

## Pricing & Availability

- Per-person and group pricing
- Tiered pricing by group size
- Seasonal pricing adjustments
- Real-time availability calendar

## Itinerary Builder

- Day-by-day itinerary editor
- Include activities, meals, and accommodation
- Media support for each day

## Booking System

- Online booking with WooCommerce
- Partial/deposit payments
- Booking notes and special requests
- Automated confirmations
