# Settings

## General Settings

- **Tour Slug** — Custom URL slug for tours
- **Tours Per Page** — Number of tours to display
- **Default Currency** — Set pricing currency
- **Date Format** — Choose date display format

## Booking Settings

- **Minimum Group Size** — Minimum travelers per booking
- **Maximum Group Size** — Maximum travelers per booking
- **Booking Cut-off** — Advance booking required (hours/days)
- **Cancellation Policy** — Configure cancellation terms

## Payment Settings

- **Deposit Amount** — Percentage or fixed deposit
- **Full Payment Deadline** — When full payment is due
- **Payment Gateways** — Available payment methods

## Email Settings

- **Booking Confirmation** — Customer confirmation template
- **Payment Reminder** — Reminder for pending payments
- **Admin Notification** — New booking alert
