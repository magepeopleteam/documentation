# WPTravelly

WPTravelly is a feature-rich travel booking plugin for WordPress, designed for tour operators, travel agencies, and trip organizers.

## Key Features

- Tour and package creation
- Multi-day trip management
- Group size and pricing tiers
- Itinerary builder
- WooCommerce payment integration
- Elementor widgets

## Getting Started

Go to **WPTravelly** in your WordPress admin to start creating tours.
