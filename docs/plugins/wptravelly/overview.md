# WpTravelly

WpTravelly is a feature-rich travel tour and booking manager plugin for WordPress, designed for tour operators, travel agencies, and trip organizers.

## Installation & Setup

Install the WooCommerce Tour & Travel Booking Manager plugin via WordPress admin, zip upload, or FTP to start creating and managing tour packages.

### Plugin Installation

- [Install the Tour & Travel Booking Manager plugin](/docs/plugins/wptravelly/installation-setup)
- Install via WordPress admin search for "Travel Booking Manager"
- Upload the zip file from MagePeople after purchasing Pro
- Ensure WooCommerce is installed and active

### Initial Configuration

- Complete the setup wizard for core tour settings
- Configure global currency and date/time formats
- Set up payment gateways through WooCommerce
- Define tour categories, tags, and organizers
