# Seasonal Pricing Addon

Offer dynamic tour pricing based on peak seasons, holidays, or special dates. Helps your travel agency boost sales and optimize revenue with smart pricing strategies.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Usage

1. Navigate to the tour edit page and find the **Seasonal Pricing** section.
2. Add a new pricing period with start and end dates.
3. Set the special price for that period.
4. Multiple seasonal periods can be added for the same tour.
