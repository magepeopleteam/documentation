# Group Ticket Quantity Addon

Sell group tickets with ease. Set group sizes and discounted rates, encouraging larger bookings while keeping your tours profitable and well-organized.

## Installation

1. Download the addon zip from your account.
2. Go to **Plugins → Add New → Upload Plugin**.
3. Upload the zip and click **Install Now**.
4. Activate the plugin from the Plugins page.

## Usage

1. Go to the tour edit page and enable **Group Ticket Quantity**.
2. Set the minimum and maximum group size.
3. Configure the group discount rate.
4. Customers can now book group tickets at the discounted rate.
