# Shortcodes

## Coupon Form

```
[adc_coupon_form]
```

Display a coupon input form.

## Coupon List

```
[adc_coupon_list limit="10"]
```

Display available coupons for customers.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| limit | int | 10 | Number of coupons to display |

## Discount Table

```
[adc_discount_table]
```

Display tiered discount pricing table.

## Offer Popup

```
[adc_offer_popup coupon_code="SAVE20"]
```

Display a coupon offer popup.
