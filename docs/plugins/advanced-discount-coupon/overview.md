# Advanced Discount Coupon

Advanced Discount Coupon is a powerful WooCommerce coupon management plugin that enables dynamic discount rules and promotional campaigns.

## Key Features

- Dynamic coupon generation
- Cart-based discount rules
- Buy one get one free (BOGO)
- Tiered pricing discounts
- Category and product-specific coupons
- Expiry date management

## Getting Started

Go to **Marketing → Coupons** in your WordPress admin to start creating advanced coupons.
